[章节标题]
剧情版本v3.2 Part 4: 影面颠倒的兔影

[章节核心目标]
- 目标1：通过异常飞讯引导漂泊者前往天台，建立与洛瑟菈的明暗双线通讯体系
- 目标2：展现学院内部对洛瑟菈的质疑，并呈现洛瑟菈从容接纳偏见的领袖气度
- 目标3：通过与塞维教授的周旋获取勋章，隔空揭穿暗面规则与教授隐瞒的苦心
- 目标4：强行突入放映厅粉碎残星会幻境，引出绯雪斩除残象的瞬间高光并完成收网

[章节前置状态]
- 地点/环境：初为漂泊者光线昏暗的个人宿舍，终端屏幕的光芒在黑暗中闪烁；后转移至拉贝尔学部顶楼天台，夜风呼啸，学院的霓虹灯光与深邃的夜空交相辉映。
- 主角状态：漂泊者刚刚结束一天的疲惫行程正准备休息，随身带着那张由千咲找回的、属于自己失忆前在星炬学院的旧学生证。
- 伴随NPC：初期独自一人，后在天台遭遇正在进行记忆调谐的洛瑟菈，随后的调查中分别遭遇卡尔教授与塞维教授。
- 指引行为：静谧的房间内，漂泊者的终端突然接连震动，竟是一向严谨端庄的洛瑟菈校长发来的飞讯，且屏幕上弹出了一个极具反差感的阿布打招呼表情包，强烈的违和感促使漂泊者立刻动身赴约。

漂泊者与洛瑟菈脱离暗面空间，在静谧的夜空下倾听完关于罗伊人老师的过往，明白探索的意义并结束对话。

:: 主线事件1：违和的飞讯与旧日学生证 ::
> 事件概况
- 漂泊者在终端上收到洛瑟菈带有阿布表情包的飞讯，调侃着一向严肃的校长居然这么活泼，随后校长吐槽并邀请他前来，漂泊者前往拉贝尔学部顶楼天台赴约。
- 漂泊者撞见洛瑟菈正独自对着虚空喃喃自语，说着什么昨晚助眠失败的音乐与罗伊族特色早餐的记忆。
- 洛瑟菈解释她刚刚是在利用自己的共鸣能力调谐记忆，她能通过物件读取记忆，但人的脑容量始终有限，所以洛瑟菈不得不定期删除部分记忆。
- 洛瑟菈为验证暗面兔子的阴谋，向漂泊者借用随身物品。漂泊者拿出了那张由千咲寻回的、记录着自己失忆前在学院过往的旧学生证。
- 洛瑟菈抚摸着这张充满岁月痕迹的旧学生证，感叹被赋予深远意义的物件能让她更容易读取附着其上的记忆频率，并借此完全看清了漂泊者在庆典暗面追逐兔子的全过程。
- 洛瑟菈推断兔子是残星会的棋子，提出两人一明一暗合作调查的计划，并将一个外表普通的化妆镜递给漂泊者，嘱咐其放在口袋中作为不被察觉的隐秘通讯器。

> 区块目标
- 目标1：利用极具反差的表情包飞讯打破洛瑟菈平日的严肃形象，拉近角色距离。
- 目标2：明确旧学生证的特殊背景，交代记忆系共鸣者读取频率的条件。
- 目标3：建立化妆镜通讯机制，为后续的隔空探案铺垫。

> 执行引导
- 描写重点：
  视觉镜头：天台微弱的月光打在旧学生证斑驳的边缘上；洛瑟菈递出化妆镜时，眼神中透着特工交接般的默契。
  互动细节：漂泊者低头看向终端上那个跳动的阿布表情包，无奈地叹了口气；随后将化妆镜滑入外套口袋，隔着布料调整到一个自然且能接收声音的角度。
- 必然触发台词/梗：
  - 漂泊者（飞讯中）：“校长怎么会使用这个表情，你是谁！”
  - 洛瑟菈（飞讯中）：“看来我们的学生对校方的老师有一种极其“微妙”的刻板印象啊。”
  - 洛瑟菈：“没想到你还保留着它，嗯……读取被赋予深远意义物件的频率，似乎也会更加容易。”
  - 洛瑟菈：“使用它的秘诀就是——保持轻松、自然。”
- 伏笔/暗线：洛瑟菈展现出的彻底删除记忆的能力，暗示了她为了背负领头羊的责任，必须不断舍弃个人的细微情感与日常碎片。

:: 主线事件2：偏见的回声与教授的勋章 ::
> 事件概况
- 漂泊者在校园内寻找塞维教授的途中，偶然目睹卡尔教授正在训斥学生，并顺势听到卡尔对洛瑟菈极其严厉的贬低，指责她无学识、无经验，根本不配背负星炬学院的未来。
- 洛瑟菈通过化妆镜通讯器清楚地听到了这些恶毒的质疑，但她没有丝毫愤怒。她平静地告诉漂泊者，身处这个位置不可避免会面临这种声音，她从容接纳了这些偏见，甚至反问漂泊者心中星炬学院的未来该是什么样。
- 漂泊者顺着线索找到塞维教授，他起初表现得极其疲惫与神经质，错把漂泊者当成助理，嘴里疯狂念叨着校准仪与炉芯数据。
- 当漂泊者以暗面受害者的同理心试探兔子的存在时，塞维教授的动作瞬间僵住。他眼神闪躲，强行压抑着恐惧，生硬地否认自己在暗面见过任何东西，并情绪激动地要求校方彻底封锁暗面。
- 塞维为感谢漂泊者在换日仪式上的贡献，将一枚停办已久的深空联合科学勋章赠予漂泊者。漂泊者走到僻静处，将口袋里的化妆镜对准勋章。
- 洛瑟菈隔空读取勋章记忆，还原了塞维在暗面被兔子变成的自己蛊惑，险些用生命去交换真理的残酷过程。从而推断出暗面异变的致命法则：利用共鸣能力对空间施加影响便会引出兔子。

> 区块目标
- 目标1：通过卡尔的漫骂与洛瑟菈的平静对比，极致凸显洛瑟菈强大的心理素质与包容质疑的领袖气度。
- 目标2：细致刻画塞维教授从狂热学者到惊恐逃避的情绪突变，展现暗面攻心幻境的恐怖。
- 目标3：完成推理闭环，促使洛瑟菈决定以身入局。

> 执行引导
- 描写重点：
  视觉镜头：卡尔教授唾沫横飞的厌恶表情；塞维教授听到“暗面”二字时猛然收缩的瞳孔和死死捏住勋章发白的骨节。
  互动细节：漂泊者在听到卡尔的谩骂时下意识想要捂住口袋里的镜子；洛瑟菈在镜子那头传来的呼吸声依旧平稳，没有任何波澜。
- 必然触发台词/梗：
  - 卡尔教授：“而她，一个平平无奇、无学识无经验的共鸣者，真的有资格背负起星炬学院的未来吗？”
  - 洛瑟菈：“没事，像那样质疑的声音，在我这个位置也是不可避免的。”
  - 塞维：“我带着它，只是想证明人类的科学曾经真的前进过……哪怕现在……”
  - 洛瑟菈：“有时候人的伟大不在于成就，而在于认清世界后，依旧坦然面对命运，有尊严地活着。”
- 伏笔/暗线：塞维教授将象征最高荣誉的勋章轻易送出，暗示他在暗面的天平上已经见证了这些荣誉被视作“毫无重量”的虚无，内心已经产生了动摇。

:: 主线事件3：幻境放映厅与赴死的羊群 ::
> 事件概况
- 洛瑟菈还有一则信息没有告诉漂泊者：自庆典结束后，暗面的受害者还有一个共同点，他们都是记忆系共鸣者，而这样的共鸣者在学院还有最后一位，洛瑟菈自己。
- 洛瑟菈知道残星会是冲着她来的，她切断了通讯，决定效仿塞维教授在暗面发动共鸣能力对空间施加影响，独自以身入局来解决暗面异变的根源。漂泊者虽不知晓真相，但担忧其安危，前往资料室在一本电影分镜册中找到备选谜题，设置好稳定出口后强行突入暗面。
- 漂泊者击溃沿途聚集的残象，深入暗面，来到一座诡异的电影放映厅，见到了正在看电影的洛瑟菈。
- 银幕上正在播放由残星会制作的黑白片《洛瑟菈的悲剧人生》，影片残忍地断章取义，重现了洛瑟菈年少时因能力失控而丧失自我、以及面对灾难自认平庸的过往。
- 兔子化作“演员·洛瑟菈”从银幕走出，企图用虚无主义诱导洛瑟菈放弃校长身份，沉溺于灾难前最后的欢愉。
- 洛瑟菈面对心理攻势冷笑出声，毫不留情地嘲讽这部电影毫无品味。她霸气地宣告，星炬学院的师生本就是一群为了真理不惧消亡、甘愿化作星炬之火的“赴死羊群”，彻底粉碎了残星会的精神压迫。

> 区块目标
- 目标1：展现残星会杀人诛心的幻境手段，将洛瑟菈的脆弱过往暴露在银幕上。
- 目标2：通过洛瑟菈的高燃演讲，完成其作为领头羊的角色弧光升华。

> 执行引导
- 描写重点：
  视觉镜头：银幕上扭曲黑白的线条如毒蛇般缠绕；洛瑟菈面对“演员·洛瑟菈”的蛊惑时，微微扬起的下巴和极具压迫感的冷冽目光。
  互动细节：漂泊者拔出武器护在洛瑟菈身侧，洛瑟菈则从容地抬手示意漂泊者无需紧张，将后背完全交给了漂泊者。
- 必然触发台词/梗：
  - ？？？：“剧本将进行重置修改，稍后将于放映厅进行试映，请演员先在幕前等待。”
  - 洛瑟菈：“不仅盗用了我的名字，还取了一个完全没有品位的标题。”
  - 洛瑟菈：“但文明的重量总得需要有人来支撑，我们本就是一群赴死的羊群。”
  - 洛瑟菈：“无需铭记，无需赞美，我们将长眠于此，不留任何痕迹。”
- 伏笔/暗线：残星会大费周章针对记忆系共鸣者，甚至称他们为“虚无的对立面”，暗示了记忆与情感是抵抗毁灭力量的核心关键。

:: 主线事件4：惊鸿一瞥的刀光与星海长谈 ::
> 事件概况
- 幻境被洛瑟菈的精神力击碎，伪装的兔子企图遁逃，四周空间震荡，大批残象涌出试图阻拦。
- 就在漂泊者准备迎战的瞬间，一道极其冷冽、霸道的寒气骤然撕裂暗面空间。神秘巫女绯雪犹如鬼魅般降临，没有一丝多余的动作，一刀便将所有残象连同空间壁垒斩成齑粉。
- 绯雪甚至没有转头看漂泊者一眼，在确保障碍扫清的半秒内，便化作一道阴影融入暗面深处，光速离场，只留下空气中残留的凌厉杀意。
- 漂泊者与洛瑟菈趁机上前捕获兔子。残星会会长借兔子之口发出预告，宣告下一场游戏将赌上拉海洛的存亡，随后兔子彻底消散。
- 两人脱离暗面返回天台。漂泊者有些埋怨洛瑟菈独自行动，洛瑟菈则告诉他：自己有十足的把握应付暗面，因为她早已能坦率地面对过去。
- 为解开漂泊者的疑惑，洛瑟菈在星光下讲述了自己曾经受到共鸣能力的不良反应影响，即将精神崩溃时，被一位老罗伊人观测者拯救的过往。
- 洛瑟菈通过老人的孤独与执着，领悟到了直面死亡恐惧的清醒，以及寻找未来意义的真谛。对于绯雪的身份，洛瑟菈以职责为由保留了悬念，两人在夜风中结束了这场特殊的“实践课”。

> 区块目标
- 目标1：以雷霆万钧之势呈现绯雪的武力值，并通过她的光速离场绝对保持其神秘感与逼格。
- 目标2：借由兔子的最后遗言，将区域危机推向最高潮。
- 目标3：在静谧的长谈中完成情绪落地，让漂泊者真正理解洛瑟菈的内心世界。

> 执行引导
- 描写重点：
  视觉镜头：绯雪拔刀时照亮暗面的惊艳寒芒，以及她收刀隐入黑暗时决绝的背影；切回现实后，两人仰望拉海洛冰原上空璀璨且冰冷的星河。
  互动细节：漂泊者被那惊艳的一刀震慑，刚刚伸出手试图挽留，绯雪便已消失无踪；洛瑟菈在讲述往事时，语气变得极其温柔，目光仿佛穿透了时空。
- 必然触发台词/梗：
  - 奇怪的兔子：“下场游戏可要赌上拉海洛这片地区的命运哦，究竟是“砰”的一声被毁灭掉，还是星炬之火仍在燃烧……”
  - 洛瑟菈：“我第一次感受到了对死亡的恐惧，我感受到了人类在那个环境下最真实的反应。”
  - 洛瑟菈：“在这个问题上，我和你们一样，都是学生中的一员。”
- 伏笔/暗线：洛瑟菈对绯雪身份的避而不谈，以及暗处绯雪弄坏了第四把刀的细节（尽管漂泊者未见），共同构成了学院“最终预案”的终极底牌。
