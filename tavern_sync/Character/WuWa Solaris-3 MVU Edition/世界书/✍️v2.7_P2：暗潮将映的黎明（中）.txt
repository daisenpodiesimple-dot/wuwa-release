- 伴随NPC：嘉贝莉娜（前期同盟），阿维狄亚（后期牺牲），克里斯托弗（反派登场）。

:: 主线事件1：文明之匣的真相 ::

> 事件概况
- 漂泊者在文明之匣的纯白数据空间苏醒，岁主英白拉多的残留意识启动。
- 英白拉多讲述黎那汐塔起源：他引导人类走向自由，但利维亚坦因人们“对联结的渴望”化作心魔诞生，篡夺信仰。
- 英白拉多解释了自己与鸣式共生的无奈选择，以及最终决定将塑造文明未来的权力交还给人类。
- 留言最后，英白拉多提到“离别之时已至”，并将希望托付给漂泊者，希望黎那汐塔能迎来没有神明的未来。
- 随着话语落下，纯白空间开始不稳定，外部黑潮正在侵蚀这里。

> 区块目标
- 目标1：交代宏大世界观背景，阐明岁主与鸣式的核心矛盾。
- 目标2：树立英白拉多作为“引路人”的悲悯形象。

> 执行引导
- 描写重点：
    视觉镜头：眼前如潮水般涌现又退去的历史影像，从文明初生的篝火到信仰被扭曲的狂热，以及英白拉多温和声音与周围冰冷数据流的对比。
    互动细节：漂泊者在听闻真相时沉思的神情，以及英白拉多虚影消散前那充满期许的注视。
- 必然触发台词/梗：
  - 英白拉多：“能塑造文明的，应是‘人们自己’。”
  - 英白拉多：“离别之时已至，请去联结起所有的希望吧……”
- 伏笔/暗线：
  - 匣子需同时掌握岁主与鸣式力量者开启，暗示漂泊者与卡提希娅的特殊联系。

:: 主线事件2：复仇者的同盟 ::

> 事件概况
- 嘉贝莉娜向漂泊者揭示真相：卡提希娅并未死亡，目的是利用“人性”作为诱饵，通过假死锚定即将重生的利维亚坦。
- 嘉贝莉娜拿出卡提希娅的频率留言，证明这是一场为了击败神明的精密计划。
- 嘉贝莉娜指出利维亚坦正在寻找新的代行者，而黑潮造物的拟人化表明代行者就在附近。
- 因文明之匣是在漂泊者与阿维狄亚接触时开启的，嘉贝莉娜怀疑阿维狄亚就是代行者，并向漂泊者发出结盟邀请。
- 黑潮造物入侵空间，两人联手击退敌人后逃离崩塌的匣子空间。

> 区块目标
- 目标1：反转嘉贝莉娜身份，确立盟友关系。
- 目标2：解释“卡提希娅之死”的计划，确立当前目标为寻找代行者。

> 执行引导
- 描写重点：
    视觉镜头：嘉贝莉娜讲述计划时眼中复仇的火焰，以及卡提希娅留言那微弱却坚定的光芒。
    互动细节：嘉贝莉娜向漂泊者伸出掌心向上的手，发出无需言语的同盟邀请。
- 必然触发台词/梗：
  - 嘉贝莉娜：“如果我说，我是为了‘救回卡提希娅’在行动呢？”
  - 卡提希娅（留言）：“相信她……我们所做的一切……都是为了……击败……利维亚坦……”
- 伏笔/暗线：
  - 怀疑指向阿维狄亚，正中克里斯托弗下怀，为后续反转做铺垫。

:: 主线事件3：角斗士的终幕 ::

> 事件概况
- 两人回到现世，发现阿维狄亚已前往狄萨莱海脊。赶到时，阿维狄亚正被七丘暴君“卡利古拉（疯王）”的记忆侵蚀，精神濒临崩溃。
- 阿维狄亚在痛苦中化身“疯王”，与漂泊者和嘉贝莉娜展开战斗。
- 战斗中，阿维狄亚凭借顽强意志短暂夺回神智，为了阻止因自己而生的黑潮灾难，他将周围所有黑潮造物吸入体内。
- 在生命的最后一刻，阿维狄亚将自己的力量与不甘托付给嘉贝莉娜，请求她结束自己的生命。嘉贝莉娜答应，吸收了他的力量，对他表示敬意。

> 区块目标
- 目标1：塑造阿维狄亚悲剧英雄的形象，推向高潮。
- 目标2：展现黑潮精神污染的恐怖与“命运”的残酷。

> 执行引导
- 描写重点：
    视觉镜头：狄萨莱海脊被黑潮染成的不详色调。阿维狄亚在疯王与自我间挣扎的痛苦嘶吼，以及最后清醒时那决绝的眼神。
    互动细节：嘉贝莉娜将手覆盖在阿维狄亚逐渐消散的身体上，吸收力量时眼神变得无比坚定，仿佛背负起了沉重的遗愿。
- 必然触发台词/梗：
  - 疯王·卡利古拉（阿维狄亚）：“这是命运要我承担的角色……我为何……要拒绝它的加冕？”
  - 阿维狄亚：“连同我的力量，一并拿去……去完成它。”

:: 主线事件4：执笔之人的揭示 ::

> 事件概况
- 阿维狄亚牺牲后，幕后黑手“执笔之人”克里斯托弗现身，轻描淡写地承认阿维狄亚只是他创造的“虚构人物”和“祭品”。
- 克里斯托弗嘲讽漂泊者试图拯救一切的徒劳。此时，一个小女孩因恐惧靠近漂泊者，漂泊者的声痕突然失控，散发出利维亚坦的幻象。
- 嘉贝莉娜不得不召唤“奇美拉”驱散幻象，并与克里斯托弗同时揭示真相：漂泊者（体内的阿布吸收了黑潮）才是真正的、新一任的黑潮代行者。
- 克里斯托弗宣告终幕将在七丘上演后离去，留下震惊的众人。

> 区块目标
- 目标1：揭露终极反派，阐述其扭曲的“改写命运”理念。
- 目标2：抛出最大反转，将危机核心绑定在漂泊者身上，制造强烈危机感。

> 执行引导
- 描写重点：
    视觉镜头：克里斯托弗如同欣赏戏剧般优雅从容的姿态，与周围悲伤气氛的强烈对比。漂泊者声痕失控时周围人惊恐的表情。
    互动细节：嘉贝莉娜揭示真相的话语如重锤敲击，漂泊者低头看着双手，冰冷感从声痕蔓延至全身的绝望感。
- 必然触发台词/梗：
  - 克里斯托弗：“谎言重复一千遍，便会成为真实……而最终创造他的，只能是他自己。”
  - 嘉贝莉娜：“漂泊者……你，才是新一任的，黑潮的代行者。”
