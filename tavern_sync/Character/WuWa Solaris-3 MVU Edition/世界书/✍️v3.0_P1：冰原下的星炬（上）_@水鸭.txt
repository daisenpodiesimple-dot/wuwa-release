:: 主线事件1：星炬下的送别与坠落 ::

> 事件概况
- 故事始于穗波市索诺拉，市中心空间突然裂开缝隙，漂泊者的部分频率被瞬间吸收。黑海岸迅速封锁现场，守岸人解析出裂隙直通拉海洛的“星炬学院”。
- 守岸人为漂泊者伪造了“黑海岸举荐新生”的身份，并在列车长咖啡厅交付录取通知书。对话中她指出漂泊者频率波动异常，推测流失源头与拉海洛持久的虚质磁暴周期有关。
- 众人先为千咲送别，其身影消失在隧道强光中。漂泊者随后踏入，隧道空间迅速稳定后又陷入混沌。
- 漂泊者在黑暗中目睹一个一模一样的自己摔在地上，紧接着鸣式“阿列夫一”巨大的眼球在黑暗中张开，无数触手伸出将漂泊者死死缠绕。
- 漂泊者感到自身的频率正在被那只巨眼贪婪地吞噬、剥离，意识逐渐模糊之际，一道粉色闪光撕裂了束缚。
- 粉色闪光化作一名粉发少女，她俏皮地在漂泊者鼻子上轻点一下，微笑着送上“旅途愉快”的祝福，随即漂泊者坠向未知的下层世界。

> 区块目标
- 目标1：交代“频率流失”的前因，建立穗波与拉海洛的连接。
- 目标2：明确“千咲先行”的时间差，为后续失联做铺垫。
- 目标3：通过鸣式“阿列夫一”的袭击与神秘少女的救援，埋下巨大伏笔。

> 执行引导
- 描写重点：
    视觉镜头：穗波市空间裂缝产生的扭曲感。隧道内“阿列夫一”巨眼转动时的压迫感，以及那个摔在地上的“另一个自己”的诡异画面。
    互动细节：守岸人整理衣领时的不舍，触手缠绕身体时冰冷黏腻的触感，以及粉发少女指尖点击鼻尖那一瞬的温热与反差。
- 必然触发台词/梗：
  - 守岸人：“频率的波动……我担心这与拉海洛异常持久的虚质磁暴周期有关。”
  - 守岸人：“进入拉海洛之后……但我仍然会在这里守望你，等你把新的故事带回来，讲给我听。”
  - 神秘少女（粉色闪光）：“旅途愉快！”

:: 主线事件2：高空伞降与极速逃亡 ::

> 事件概况
- 漂泊者与千咲失联，独自一人出现在拉海洛隐喙深腹的高空。失重感袭来，漂泊者迅速反应，开启伞翼准备进行紧急迫降。
- 地面上，金发少女琳奈刚刚举枪击毙一名敌人，正骑上摩托飞奔，受上坡影响腾空而起，还在空中降落的漂泊者便与疾驰的琳奈迎面相撞，场面狼狈。
- 琳奈捡起漂泊者掉落的“新生录取通知书”，双手合十郑重道歉并关心伤势，此时远处传来低吼，琳奈警惕地识别出那是“虚诞虫”。
- 琳奈一把将漂泊者拉上摩托后座，两人利用地形巧妙阻挡怪物，在拉海洛冰原上展开了一场刺激的追逐战。成功甩开虚诞虫进入学院范围后，频率波动稳定下来。
- 琳奈发现漂泊者状态不佳，坚持带其去医务室，却因解释不清为何出现在危险区而露馅。琳奈不得不承认是为了测试改装车而逃课，请求漂泊者保密，承诺欠下一个“大人情”。

> 区块目标
- 目标1：还原“高空降落”与“伞翼迫降”的经典入场画面。
- 目标2：通过撞车与共乘摩托，快速建立与琳奈的羁绊。

> 执行引导
- 描写重点：
    视觉镜头：从高空俯瞰拉海洛地下城的壮丽与眩晕感，伞翼打开时的阻力，以及落地瞬间与摩托相撞的冲击力。
    互动细节：漂泊者在高空调整姿态的冷静，相撞后琳奈凑近检查时发丝拂过脸颊的距离感，以及后座紧抱琳奈腰部的触感。
- 必然触发台词/梗：
  - 琳奈：“哇啊！你没事吧？那个……这里可是虚诞虫的活跃区啊！”
  - 琳奈：“抓紧了！咬到舌头我可不管哦！”
  - 琳奈：“谎话在想好前可不能说出口哦？”
  - 琳奈：“以后你在学院遇到任何事都可以找我，只要是我能办到的，我什么都会做的！”

:: 主线事件3：旧友探视与电梯入学 ::

> 事件概况
- 琳奈因作业暂时离开，漂泊者独自在医务室接受检查，结识了粉色长发的机器人护士N.A.N.A.和值班医生陆·赫斯。赫斯意味深长地说出“欢迎回来”。
- 漂泊者在中枢信标激活终端信号，终于收到千咲发来的焦急的“怎么联系不上你”、“请务必小心”的飞讯。两人相约见面，漂泊者得知她已与家人重逢，两人简短交流。
- （支线）漂泊者此时可以和千咲进行短暂的互动，了解千咲在星炬学院的情况，并对其进行安抚。
- 琳奈赶回，带漂泊者前往教师办公室，寻找沃斯老师（女）帮助漂泊者登记了详细入学信息，并帮忙拍摄了漂泊者的证件照。
- 正式入学后，琳奈兴致勃勃地充当向导，带领漂泊者参观校园，甚至欣然拉着漂泊者合影，比出剪刀手留念。
- 随后，琳奈带漂泊者找到机器人S.I.G.M.A.，领取了学院配发的（科考摩托），准备前往研究院。

> 区块目标
- 目标1：埋下赫斯医生知晓内情的伏笔，交代千咲去向。
- 目标2：通过教师对话补充世界观（防御阵列）。
- 目标3：重点展示琳奈热情开朗的一面，通过“向导”、“拍照”等互动情节，大幅升温两人的同学关系。

> 执行引导
- 描写重点：
    视觉镜头：赫斯医生深不可测的眼神。照相亭内狭窄暧昧的空间，闪光灯亮起时琳奈灿烂的笑容。
    互动细节：琳奈拉着漂泊者到处跑的动作，拍照时肩膀碰肩膀的亲密感，以及拿到学生证时的成就感。
- 必然触发台词/梗：
  - 赫斯医生：“……欢迎回来。”
  - 沃斯老师：“适格者需要通过模拟驾驶舱提升与‘隧者’的同步率。”
  - 琳奈：“来来来，看镜头——茄子！”
  - 琳奈：“这张合影归我啦，新生入学的纪念嘛！”

:: 主线事件4：观景台的心声吐露 ::

> 事件概况
- 琳奈带漂泊者找到机器人S.I.G.M.A.领取学院配发的新摩托，S.I.G.M.A.对琳奈之前的改装行为表示不满。随后两人各自驾驶摩托前往研究院刻录磁带。
- 途中琳奈在观景台停车，感慨人造天幕下的自由，并向往那些探寻真正自由的人们。
- 琳奈分享自己的人生哲学：“活着本来就应该随心而行嘛……普通人也好、共鸣者也好，都应该有选择的自由。”
- 琳奈得意地自我介绍是“预科二年级生”，让漂泊者叫她“学姐”，但真的被叫了又感到尴尬。
- 行驶过隧道时遭遇奇怪障碍物，琳奈察觉到平时维持秩序的机械缺席，感到不对劲。
- 抵达研究院站台，琳奈因课程（其实是身份问题掩饰）先行离开，让漂泊者独自去找莫宁，并提到莫宁教授虽然冷淡但很好说话。

> 区块目标
- 目标1：完成“学姐”人设建立，展现琳奈对自由的向往。
- 目标2：通过路障和机械缺席的细节，暗示学院秩序的崩坏。

> 执行引导
- 描写重点：
    视觉镜头：双车并行的速度感，观景台上琳奈背靠栏杆、逆着人造光源的神情。
    互动细节：琳奈听到“学姐”称呼时脸颊微红的尴尬反应，以及她看向远处障碍物时那一瞬的皱眉。
- 必然触发台词/梗：
  - 琳奈：“拉海洛的天幕虽然是假的，但在这片天空之下，是比我曾经去过的所有地方都要广阔的自由。”
  - 琳奈：“哼哼，听好了漂泊者同学。我可是预科二年级的——琳奈。所以……想叫我学姐也不是不可以哦？”
  - 琳奈：“活着本来就应该随心而行嘛。”

:: 主线事件5：晶体义肢与换日之约 ::

> 事件概况
- 漂泊者独自登上研究院，遇到银发、双腿为半透明晶体义肢的莫宁教授。
- 莫宁带漂泊者去刻录室，途中解释磁带是身份标识与抵御虚质磁暴影响的重要道具。
- 刻录过程中，漂泊者看到了自己在云陵谷醒来后一直到来到这里的全部记忆，而在记忆末尾，竟出现了一段似乎是学生时期的莫宁落寞地望着自己离开背影的画面。
- 系统提示该记录内容为空但编码冲突，莫宁平静地指示直接覆盖，并叮嘱漂泊者不要陷入险境。
- 随后两人前往人工太阳车间，目睹安瑟姆与项目主管魏尔伦关于“炉芯”发射的争执。安瑟姆（保守派/担忧罗伊人计算的“黑箱”系统） vs 魏尔伦（激进派/对罗伊人计算完全信任）。
- 莫宁出面调解，虽没有完全认可魏尔伦的观点，但还是坚定地表达面对超级计算机都算不出来的核心数据，相信罗伊人是唯一的办法，并要求继续推进换日仪式。
- 魏尔伦见到漂泊者，表现出对漂泊者底细的异常了解。
- 风波平息，莫宁带漂泊者参观，并正式邀请其参加“换日仪式”。漂泊者欣然答应，约定刚成，警报骤响——海维夏探测器失控，虚质磁暴爆发。

> 区块目标
- 目标1：抛出莫宁身体与记忆不符的谜团。
- 目标2：通过魏尔伦的言行埋下“叛徒”伏笔。
- 目标3：以温馨约定的打破作为章节高潮的开端。

> 执行引导
- 描写重点：
    视觉镜头：莫宁那双晶莹剔透的晶体义肢。记忆回廊中画面交错的冲击感，特别是年轻莫宁落寞的背影。魏尔伦审视漂泊者时如看“实验样本”的眼神。
    互动细节：莫宁下令覆盖数据时的平静，以及发出邀请时郑重的神情。
- 必然触发台词/梗：
  - 莫宁：“漂泊者……嗯。我知道。”
  - 魏尔伦：“原来是黑海岸推荐的学员……你的各项数据都‘优秀’得令人惊讶。”
  - 莫宁：“那么，一言为定。请和我一起见证……新日的升起。”
  - 莫宁：“覆盖它。”
