[章节标题]
剧情版本v1.0 Part 1: 行至海岸尽头（上）

[章节核心目标]
- 与秧秧和新角色釉瑚汇合，了解她们在归墟港市的行动目的。
- 引导漂泊者接触黑海岸成员妮娅，并接受调查失联收容员的委托。
- 遭遇名为“霍德”的异常频率聚合体，并被强制传送，以此触发场景转移。

[章节前置状态]
- 地点/环境：归墟港市的废墟边缘。天空病态灰白，海风穿过破碎的建筑钢筋，发出呜咽声。
- 主角状态：漂泊者刚刚抵达这片死城，脚踩着碎石，怀中携带着秋水给予的黑花。
- 伴随NPC：秧秧正在不远处焦急地张望，等待着漂泊者。

[章节事件列表]
1. 主线事件：废墟重逢与暧昧约定
2. 主线事件：黑海岸的委托
3. 主线事件：凝固的城市悲鸣
4. 主线事件：引力源“霍德”与强制传送

[章节终止条件]
漂泊者的视线被强烈的蓝光吞没，与秧秧和釉瑚的呼喊声被切断，身体被无法抗拒的吸力卷入漩涡，场景完全切换。

---

:: 主线事件1：废墟重逢与暧昧约定 ::

> 事件概况
- 漂泊者踩着碎石走向废墟深处，秧秧听到脚步声猛地回头，眼中瞬间迸发出喜悦的光亮。
- 陌生少女釉瑚抛掷着手中的铜钱登场，挺起胸膛摆出“行家里手”的姿态自我介绍。
- 秧秧侧身向漂泊者解释釉瑚的身份，以及此次前来是为了寻找黑海岸以调查漂泊者身世的线索。
- 漂泊者轻声回应“没想到你还记得”，秧秧闻言脸颊迅速泛红，眼神有些慌乱地游移。
- 釉瑚在一旁发出响亮的“啧啧”声，凑近观察两人，调侃秧秧对漂泊者“特别上心”。
- 漂泊者从怀中取出那朵格格不入的黑花展示，釉瑚看到宝物后眼睛发亮，立刻凑近端详。
- 釉瑚拍着胸脯宣称要“保驾护航”，强行加入了队伍，三人开始一同行动。

> 区块目标
- 目标1：展现秧秧对漂泊者特别的“上心”，以及漂泊者对这份心意的回应。
- 目标2：借釉瑚之口打破尴尬，确立三人组队模式。

> 执行引导
- 描写重点：
  视觉镜头：秧秧见到漂泊者时眼中迸发的光亮；秧秧听到回应时脸颊泛红、手足无措的羞涩神态。
  互动细节：釉瑚抛掷铜钱的动作；釉瑚看热闹不嫌事大的表情；废墟环境的萧瑟与三人互动的温馨形成的对比。
- 必然触发台词/梗：
  - 釉瑚：“哎哟哟，原来秧秧这么在乎你呀？还是该说她对你特别上心？”
  - 秧秧：“我、我只是觉得这件事对你很重要……”
- 伏笔/暗线：
  - 釉瑚提到从乘霄山流出的“看不出属地的物件”，这为她背包里的关键道具——寄存器卡片埋下伏笔。

:: 主线事件2：黑海岸的委托 ::

> 事件概况
- 三人在混凝土迷宫中穿行，抵达架设着精密仪器的黑海岸临时营地。
- 执花妮娅正对着终端屏幕眉头紧锁，手指飞快敲击键盘，嘴里念念有词。
- 漂泊者上前搭话，妮娅猛地抬头警惕，视线触及漂泊者的金瞳和黑花后转为惊讶。
- 妮娅误以为漂泊者是安可提到的新成员，慌乱地鞠躬道歉并请求协助。
- 妮娅指着屏幕上呈现诡异紊乱波形的监测仪，解释“异常频率”可能引发新一轮悲鸣。
- 妮娅神色黯淡地提及收容员失联，漂泊者与秧秧对视一眼，秧秧握紧剑柄主动请缨。
- 妮娅将异常坐标传输给漂泊者，并郑重告诫该区域存在严重的重力异常。

> 区块目标
- 引出本章的核心任务，交代“异常频率”和“重力异常”的关键线索。
- 塑造黑海岸成员专业、专注但略带冒失的形象。
- 强化漂泊者“大名鼎鼎”的外部声望。

> 执行引导
- 描写重点：
  视觉镜头：临时营地的科技感；终端屏幕上诡异紊乱的波形；妮娅面对终端时紧锁的眉头。
  互动细节：妮娅见到漂泊者后从警惕到惊讶再到慌乱的情绪转变；秧秧听到严重事态时握紧剑柄的动作。
- 必然触发台词/梗：
  - 妮娅：“诶！？可是安可说，你很快就会加入黑海岸的……”
  - 妮娅：“那段异常频率近似于一种引力源，在它附近应该能很明显地感受到重力异常。”
- 伏笔/暗线：
  - 首次将“异常频率”与“悲鸣”以及“归墟港市的毁灭”联系起来，暗示其危险性远超预期。

:: 主线事件3：凝固的城市悲鸣 ::

> 事件概况
- 三人深入坐标区域，漂泊者感到脚步变得沉重，周围的碎石开始违背物理常识悬浮在半空。
- 秧秧伸手拦住想要冲锋的釉瑚，示意前方出现的并非实体怪物。
- 众多半透明的模糊人影在废墟中机械地重复动作（看表、等车、闲聊）。
- 空气中传来重叠的低语声，一个学者影子的抱怨声和一对母子的对话清晰传入漂泊者耳中。
- 秧秧冲向几具倒在地上的黑海岸制服身影，检查后缓缓放下手，沉痛地摇头。
- 釉瑚捂住嘴发出惊呼，漂泊者环顾四周，感受到令人窒息的重力压迫感。

> 区块目标
- 营造一种混合了诡异、悲伤与恐怖的独特氛围。
- 具象化地展示“悲鸣”的恐怖之处——对逝者日常的永恒囚禁。
- 证实妮娅的不祥预感，将事件的严重性升级。

> 执行引导
- 描写重点：
  视觉镜头：残响呈现为半透明、模糊的人影形态；碎石悬浮的画面；收容员尸体的特写。
  听觉镜头：空气中回荡着断断续续、重叠在一起的日常低语（等地铁的母子、聊游戏的青年等），这种日常感与死亡废墟的反差是关键。
- 必然触发台词/梗：
  - 母亲影子：“下一班地铁要十分钟之后，很快啦。”
  - 学者影子：“不敬畏头顶星空的人，终究会被时代抛弃。”
- 伏笔/暗线：
  - 收容员的死亡证明了此地存在着能轻易杀死黑海岸专业人员的未知危险。

:: 主线事件4：引力源“霍德”与强制传送 ::

> 事件概况
- 几道虚幻影子在尸体旁凝聚，化为一个名为“霍德”的女性轮廓，发出失真的合成音。
- 周围所有的残象（母子、青年、学者）齐齐转头，空洞的面孔死死盯着漂泊者。
- 霍德张开双臂，周围悬浮碎石剧烈震颤，她尖啸着要求漂泊者想起作为黑海岸一员的过去。
- 霍德的影子突然剧烈扭曲，语气突变，指向釉瑚大喊“快跑”。
- 釉瑚背包震动，一张看似普通的卡片自行飞出，爆发出肉眼可见的黑色波纹。
- 黑色波纹扫过，周围的碎石瞬间化为齑粉，空间像破碎的镜面一样裂开。
- 巨大的吸力产生，釉瑚惊叫着被吸向漩涡，秧秧伸手死死抓住她。
- 漂泊者抬头，发现头顶灰暗的天空瞬间被深邃星空取代，黑色花瓣如雨落下。
- 温柔的女声直接在脑海响起，强烈的蓝光吞没视线，漂泊者感到失重并坠入深渊。

> 区块目标
- 剧情推向本章高潮，揭示核心谜团“霍德”的存在。
- 引爆伏笔，证实釉瑚身上的卡片是关键道具。
- 以强大的外部力量强制结束当前场景，并用神秘声音“欢迎回家”制造巨大悬念。

> 执行引导
- 描写重点：
  视觉镜头：霍德影子的虚幻与扭曲；所有残象转头的惊悚瞬间；卡片爆发黑色波纹将碎石粉碎的视觉冲击；天空变为星海的超现实景象。
  听觉镜头：霍德那由无数人声合成的失真声音；脑海中清晰而温柔的神秘女声。
- 必然触发台词/梗：
  - 霍德：“全都想起来吧，你的过去……你割舍的过去……你作为黑海岸一员的过去……”
  - 神秘声音：“欢迎回家。”
  - 神秘声音：“无论你到了哪里，瑝珑也好，黎那汐塔也好，我都会在这里等你。”
- 伏笔/暗线：
  - 霍德提及“你作为黑海岸一员的过去”，直接点明漂泊者与该组织有深刻渊源。
  - 霍德最后指向釉瑚并提及“就在她身上”，暗示了卡片是找到真相的关键。
