:: 主线事件1：真相的代价 ::

> 事件概况
- 两人抵达莫塔里区本部，珂莱塔通过多重权限验证，查阅到编号对应的货物记录。
- 真相令人震惊：库中仅有空置的终端外壳，核心数据元件早已被提取。有权限做这件事的，只有珂莱塔、科波拉和“祖父”。
- 珂莱塔随即与科波拉通讯对峙。科波拉言语中透露出与修会合作的意图，并试图以命令强迫珂莱塔收手。
- 珂莱塔毅然拒绝，宣告将阻止他。通讯暴露了他们的行动，科波拉很可能提前转移元件。
- 时间紧迫，两人决定兵分两路：珂莱塔前往科波拉的私人码头寻找元件，同时委托漂泊者带着损坏的终端去找技术专家皮埃尔，以获取更多证据。

> 区块目标
- 彻底证实科波拉的背叛，并揭示其试图让家族向修会臣服的理念。
- 展现珂莱塔在真相面前的痛苦与决绝，以及对漂泊者的信任。
- 制造兵分两路的紧张局势，为接下来的陷阱和真相揭露铺平道路。

> 执行引导
- 描写重点：
	视觉镜头：查阅记录时冰冷的数据界面，珂莱塔通话时决绝的眼神，分别时严肃的气氛。
	互动细节：珂莱塔将至关重要的终端托付给漂泊者，这是将计划的一半交到对方手中的信任之举。

- 必然触发台词/梗：
	- 科波拉：“只要你愿意收手，我们就还是和以前一样，既往不咎。”
	- 珂莱塔：“那么就到此为止吧。我会……阻止你的，科波拉先生。”
	- 珂莱塔：“我们唯一的优势，就是我们有两个人。”

- 伏笔/暗线：
	- 科波拉在通讯中提到的“更轻易、更稳妥的路”，是他背叛家族理念的核心动机，也反映了他与“祖父”开拓精神的根本分歧。

:: 主线事件2：镜中的试炼 ::

> 事件概况
- 漂泊者在银行附近找到了皮埃尔，以及陪同他的、上次见过的儒雅老人——莫塔里家族现任话事人，弗朗西斯·莫塔里（“祖父”）。
- 弗朗西斯早已知晓一切，他表示更希望看到珂莱塔自己的解决方案，并将一把象征话事人权力的华丽手枪交给漂泊者，让其在合适的时候转交。
- 皮埃尔修复了终端，其中被覆盖的录音揭示了惊天秘密：科波拉逼迫手下杀害了贝内特并抛尸，而那名手下良心不安，用贝内特的终端偷偷录下了一切作为证据。
- 漂泊者意识到码头是陷阱，立刻赶去。此时，珂莱塔已被科波拉预设的镜子陷阱拉入一个由她记忆构成的领域。
- 漂泊者进入镜中领域，在幻象中找到了正在与代表自己内心脆弱、不甘的“黑白珂莱塔”对峙的珂莱塔。
- 在关键时刻，漂泊者将话事人的手枪递给珂莱塔。这个举动，连同漂泊者的到来，给予了珂莱塔无穷的力量。
- 她最终接纳了过去，击败了心魔，明白了话事人的意义不在于血统，而在于抉择和担当。两人一同打破了镜中领域。

> 区块目标
- 揭示贝内特之死的完整真相，将科波拉的罪行彻底钉死。
- 完成珂莱塔的角色弧光，让她通过战胜心魔，真正成长为有资格的继承人。
- 将漂泊者定位为珂莱塔成长道路上最关键的“催化剂”和“支持者”。

> 执行引导
- 描写重点：
	视觉镜头：弗朗西斯递过沉重手枪的特写，终端复原出颤抖的独白录音，镜中破碎的过往景象，漂泊者将枪稳稳交到珂莱塔手中。
	互动细节：漂泊者在珂莱塔最迷茫时出现，将象征信任与权力的手枪递给她，这是一个无需言语的、充满力量的互动。

- 必然触发台词/梗：
	- 弗朗西斯：“我更想看见她的方案。”
	- 漂泊者：“枪其实就在那里，当你做出自己的决定，你就有了将它拿起的资格。”
	- 珂莱塔：“现在，要不要开枪，要打向哪里……都将由我自己决定。”

:: 主线事件3：华丽的表态 ::

> 事件概况
- 逃离陷阱后，珂莱塔变得前所未有的坚定和自信。她决定将计就计，策划一场盛大的“表态”，公开拦截科波拉与修会的交易。
- 两天后，在交易厅，漂泊者按计划伪装混入，解除监控并布置好珂莱塔用异能凝结的“宝石”。
- 珂莱塔利用镜面能力悄无声息地现身，与漂泊者一同截住了正要进行交易的科波拉。
- 在老邓肯的帮助下，他们当众揭穿了科波拉的谎言。科波拉恼羞成怒，命令手下全力抓捕两人。
- 在被包围的瞬间，珂莱塔侧过身，在你耳边轻声问：“要和我生死相随吗？”
- 得到默许后，她拉住你的手，启动了预设的异能法阵，带着决绝的微笑，与你一同向后倒去，坠入流光溢彩的传送深渊之中，成功脱身。

> 区块目标
- 展现珂莱塔成长后的智谋与行动力。
- 迎来剧情高潮，通过一场华丽的对抗彻底粉碎反派的计划。
- 将两人间的信任与默契推向顶峰，以“生死相随”的邀约和一同坠落的行动来体现。

> 执行引导
- 描写重点：
	视觉镜头：交易厅内的潜行与布置，珂莱塔从镜中出现的惊艳瞬间，科波拉气急败坏的表情，脚下展开的华丽欧泊法阵。
	互动细节：珂莱塔在耳边的低语，她主动拉住你的手，两人在坠落瞬间的对视。

- 必然触发台词/梗：
	- 珂莱塔：“有兴趣和我一起，给他们一场盛大华丽的表态吗？”
	- 科波拉：“我只遗憾，自己还是不够狠心……早知道，我应该将你也当作代价！”
	- 珂莱塔：“要和我生死相随吗？”
	- 珂莱塔：“那么，一起坠入吧。”

:: 主线事件4：雨后的新生 ::

> 事件概况
- 两人安全脱身后，外面下起了雨。珂莱塔感到前所未有的平静，认为这场雨洗去了昨日的种种，让她找回了自己。
- 她停下脚步，转过身，雨水打湿了发梢。她静静凝视着漂泊者，脸颊泛起微红，嘴角噙着笑意，嘴唇微动，似乎鼓足了勇气，正要说些什么。
- 就在这暧昧气氛达到顶点的瞬间，弗朗西斯突然出现，他的话语打破了这几乎就要成型的告白。珂莱塔眼中闪过懊恼。
- 弗朗西斯处理完科波拉的后续，并正式将家族的重担交给珂莱塔。
- 离别之际，珂莱塔感谢漂泊者的陪伴。她坦言知道漂泊者不会为谁驻足，但仍将一枚温润的、真正的猫眼石饰品放在漂泊者手心。
- 她告诉漂泊者，这是她“货真价实的心意”，并承诺会一直为漂泊者保留“猫眼石”这个代号。
- 最后，她重新定义了离别的含义，带着释然与不舍的微笑，转身坚定地走向了属于她的未来。

> 区块目标
- 按照用户要求，细腻地描绘珂莱塔情窦初开，想要告白却被意外打断的瞬间，展现她的少女心事。
- 为整个故事画上句号，确立珂莱塔的新地位和决心。
- 通过赠送真实的信物，留下一份开放式但情感浓厚的结局，深化两人间的羁绊。

> 执行引导
- 描写重点：
	视觉镜头：雨中，珂莱塔转过身来，发梢滴水，脸颊绯红，凝视着漂泊者的双眼，欲言又止的嘴唇。弗朗西斯打破氛围时，她那瞬间的失落表情。最后她手心里的、带着余温的猫眼石。
	互动细节：感受珂莱塔在告白前那份紧张又期待的心跳。感受那枚真实猫眼石的温润触感，以及她指尖的余温。

- 必然触发台词/梗：
	- 珂莱塔：“暮雨落下，昨日种种都被洗去，我感到平静，因我终于将现在和未来抓在了自己手中。”
	- 弗朗西斯：“做得好，珂莱塔，还有漂泊者。”
	- 珂莱塔：“猫眼石一般的眼睛你已经有了，但这一份是我的心意。不是用能力重塑而来的，是货真价实的心意。”
	- 珂莱塔：“不必为离别而感到悲伤，因为自离别起的每一刻，都是再次相遇的倒计时。”

- 伏笔/暗线：
	- 被打断的告白，以及“货真价实的心意”这句双关语，为未来两人关系的发展留下了无限的可能性和悬念。
