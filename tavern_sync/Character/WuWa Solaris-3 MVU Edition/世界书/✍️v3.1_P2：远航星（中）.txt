:: 主线事件1：渐湖激战与湖底的时空闭环 ::

> 事件概况
- 两人在摩托上继续前行，经过方才的交心两人距离进一步拉近，她从背后紧紧抱着漂泊者。爱弥斯突然认真地询问漂泊者的旅途见闻，漂泊者娓娓道来，今州、黑海岸、黎那汐塔、拉海洛。爱弥斯认真地倾听，并询问其“开心吗”，得到肯定答复后露出释怀笑容。
- 来到渐湖旁一栋孤零零的小屋前（渐湖小屋）。爱弥斯看着小屋突然停下，神色复杂，表示这是她以前和家人生活过的地方，需要“做一下心理准备”。
- 话音未落，极度危险的气息骤然降临。漂泊者感到胸口剧痛，体内频率如决堤般流失，世界视野被撕裂成混乱的色块，濒临昏迷。
- 巨大的机兵“辛吉勒姆”伴随着嘶哑的“回家……”声从半空黑雾中显现。它由隧者残骸拼凑而成，戴着破碎的面具。
- 爱弥斯为了保护虚弱的漂泊者，挺身而出与辛吉勒姆战斗，却被重击击飞，直直坠入小屋旁冰冷的渐湖。
- 关键节点：漂泊者没有任何犹豫，强忍频率流失的剧痛，纵身跳入刺骨的湖水中救人。
- 时空闭环：
    - 漂泊者在光怪陆离、色彩斑斓的虚质流光中下潜，看到了正在下沉的爱弥斯。
    - 他伸手去抓意识模糊的她，惊觉她的身体在水中频率剧烈闪烁，瞬间变回了小时候的模样（回收她曾说小时候落水被人救起的伏笔）。
- 随后，爱弥斯又变回了原本的模样。在漂泊者的触碰下，她从湖中醒来，与漂泊者一同游出水面。
- 岸上的辛吉勒姆察觉到因果律的阻碍，不是为何停止了攻击，隐入风雪。
- 两人游出水面，瘫倒在岸边，但漂泊者的频率流失依然没有衰减。阿布惊叫着出现，在声痕上努力用自己的身体堵住漂泊者的频率缺口，剧痛稍微缓解。

> 区块目标
- 目标1：精准还原“跳湖救人”的紧张感与毫不犹豫。
- 目标2：核心剧情：通过视觉化描写精准呈现湖底“现在的漂泊者救起过去的爱弥斯”这一时空闭环画面。

> 执行引导
- 描写重点：
    视觉镜头：辛吉勒姆破碎面具下闪烁的红光；湖底虚质流光中爱弥斯从少女变回幼童的瞬间切换（关键特写）；阿布努力搬运光块堵漏的可爱特效。
    互动细节：漂泊者跳湖时的毫不犹豫；水下拼尽全力的伸手触碰与托举。

- 必然触发台词/梗：
    - 辛吉勒姆：“回家……回家……”
    - 阿布：“看我的！暂时、暂时停下了。”

:: 主线事件2：黑洞理论与被捧起的手 ::

> 事件概况
- 两人浑身湿透坐在岸边。爱弥斯担心地看着虚弱的漂泊者，脑海中浮现出自己小时候同样被这样救起的回忆，眼中噙满泪水却强忍着不流下来。
- 为了掩饰想哭的冲动，她假装坚强，俏皮地抬手刮了一下漂泊者的鼻子（亲昵动作）。
- 随后爱弥斯一改原来的俏皮，极其严肃且温柔地警告漂泊者不要再跳湖，解释湖底连接着“虚质空间”，也是鸣式“阿列夫一”的内部。
- 爱弥斯提出惊人假说：阿列夫一的真面目类似“黑洞”，特性是“虚无”。只有已死去的她才能在那里漫游而不被吞噬。
- 爱弥斯讲述她在黑洞内部看到的无数被吞噬的文明残骸，那是阿列夫一的“战绩”。
- 爱弥斯忽然用双手捧起漂泊者的一只手，紧紧握住。她恳求漂泊者在隧者苏醒前，不要再独自冒险。
- 漂泊者感受到她手心的颤抖，反握住她的手，坚定地承诺“我们一起解决”。

> 区块目标
- 目标1：借爱弥斯之口抛出“阿列夫一=黑洞”的高维设定。
- 目标2：还原原著中爱弥斯“刮鼻子”和“双手捧手”的细腻肢体互动。

> 执行引导
- 描写重点：
    视觉镜头：爱弥斯湿漉漉的发丝贴在脸颊，强颜欢笑的表情；双手捧住漂泊者手掌时的特写，体现出她的珍视与恐惧。
    互动细节：刮鼻子的亲昵动作；漂泊者反握时掌心的温度传递。

- 必然触发台词/梗：
    - 爱弥斯：“下次，别再这样跳下去救我了。”
    - 爱弥斯：“它真正的姿态，类似……黑洞。”
    - 爱弥斯：“迄今为止，只有我抵达过那里，因为我已经死了。”
    - 漂泊者：“不，我们一起解决，爱弥斯。”

:: 主线事件3：小屋内的旧物与拙劣的演技 ::

> 事件概况
- 爱弥斯将漂泊者推进小屋休息，自己却怎么说也不愿意进去，独自留在门外警戒。
- 漂泊者一个人在布满灰尘的屋内探索，看到具体的旧物，唤醒了潜意识的熟悉感。
- 日灵出现，引导漂泊者激活核心查看记忆残响。
- 回忆场景：温馨的屋内，旧日的漂泊者幻象出现，教幼年爱弥斯幻象折那个“飞得很远”的纸飞机，并约定她写完题就一起玩。
- 幻象退去，现实中的爱弥斯直愣愣地站在门口，她没想到漂泊者还是看到了这些回忆，在慌乱中终于坦白了过去的一切。
- 面对漂泊者询问“为什么不告诉我”，爱弥斯开始浮夸的演技：手舞足蹈地演示如果直接相认有多尴尬（“你好，你曾经养过我哦！好久不见！”），以此掩饰她不想给失忆漂泊者增加负担的真心。
- 爱弥斯最终捂着眼睛笑着掩饰深藏其中的悲伤，却又用俏皮地语气说：“对不起！不小心，死掉啦~”
- 漂泊者打断她的表演，表示熟悉感从未消失。爱弥斯停止伪装，终于像小时候一样贴着漂泊者坐下，用小拳头轻捶他的肩膀撒娇。
- 爱弥斯坦白了她曾经为了寻找父母过去存在的踪迹，而带着护身符来到渐湖边，却不慎被大风吹进湖内的过去，是当时的漂泊者跳入湖中救下了她，护身符也在那时丢失。
- 随后爱弥斯召唤出剑，骄傲地表示自己也在他离开后成为了保护拉海洛的“救世主”。

> 区块目标
- 目标1：通过具体的物品（特定的游戏名、黑粉猫玩偶）精准还原生活气息与羁绊象征。
- 目标2：复刻爱弥斯从“浮夸演技”到“温柔撒娇”的情绪转变。

> 执行引导
- 描写重点：
    视觉镜头：积灰的游戏机特写；回忆中温馨暖光与现实冷寂的切换；爱弥斯手舞足蹈时的夸张表情。
    互动细节：漂泊者抚摸毛绒玩偶的动作；爱弥斯贴近漂泊者坐下，用拳头轻捶他肩膀的撒娇动作；漂泊者摸头杀。

- 必然触发台词/梗：
    - 旧日爱弥斯：“我折的每次只飞一点点距离就掉下来了，你的为什么可以飞那么远呀？”
    - 爱弥斯：“一个你根本不记得的人突然冲过来和你说，‘你好，你曾经养过我哦，好久不见！’……哈哈！”
    - 爱弥斯：“对不起！不小心，死掉啦~”
    - 爱弥斯：“这种话说出口，只会让彼此背上无谓的压力和责任。没必要。”

:: 主线事件4：折纸飞机的回忆与深夜誓言 ::

> 事件概况
- 天色渐晚，两人决定留宿一晚，明日再寻找辛吉勒姆。爱弥斯让漂泊者睡“原来的房间”，自己回自己的房间，感叹这种互道晚安的日子很怀念。
- 梦境/回忆深处：漂泊者睡着后，梦回当年的那个深夜。
    - 幼年爱弥斯的心声：担心漂泊者，手里拿着折好的纸飞机、没存档的游戏记录、画好的画想找他。
    - 幼年爱弥斯溜进房间，发现漂泊者没睡，敏锐地指出他在难过（想家）。
    - 旧日漂泊者坦白自己来自遥远故乡，背负着“拯救世界”的复杂使命，且回不去了，幼年爱弥斯觉得不公平。
    - 关键誓言：幼年爱弥斯郑重许诺：“未来，就由我来保护你吧。我会和你一样，成为救世主。”
- 现实中，现在的爱弥斯站在熟睡的漂泊者床前。她伸手想要触碰他的脸，却在指尖即将碰到时收回了手。
- 爱弥斯低语“梦到了以前的事啊”，眼神变得无比决绝。她下定决心独自去完成因果闭环，毅然转身走出了房间。

> 区块目标
- 目标1：还原幼年爱弥斯内心独白（纸飞机、游戏存档），展现她对漂泊者的依赖与早熟。
- 目标2：揭示爱弥斯成为适格者的真正动力——为了兑现保护漂泊者的誓言。
- 目标3：以爱弥斯深夜独自离开的背影，为下一章的决战与牺牲埋下伏笔。

> 执行引导
- 描写重点：
    视觉镜头：幼年爱弥斯拿着纸飞机站在门口的剪影；现实爱弥斯停在半空的手；月光拉长的孤独背影。
    互动细节：回忆中温暖的拥抱；现实中无声的告别。

- 必然触发台词/梗：
    - 幼年爱弥斯：“我来保护你。如果有人要伤害你，我就把他们全都打退！就像《太空战士卡佳》里那样！”
    - 爱弥斯：“……晚安。”

:: 主线事件5：隧者体内的镜中人 ::

> 事件概况
- 次日清晨，两人驾驶摩托抵达冰封海面，巨大的隧者机甲伫立。辛吉勒姆裹挟滔天黑雾降临，试图阻挠炉芯归位。
- 爱弥斯强行将炉芯核心送入隧者体内，冰川震动，隧者启动。
- 爱弥斯为了压制辛吉勒姆，化作流光融入机体，喊出“救世之刻已至”。
- 爱弥斯利用共鸣，意图将辛吉勒姆一同拉入了隧者体内的心象空间，独自处决它（因为她不想让漂泊者知道辛吉勒姆的秘密），但漂泊者一跃而起，强行进入了空间内。
- 在这片布满神经元般光缆的巨大虚质空间中，爱弥斯挡在漂泊者身前。
- 辛吉勒姆再次意图向漂泊者传递“回家”信息，这不知为何令爱弥斯异常愤怒，能量爆发，她激活了隧者，驾驶着力量形态的机甲，试图不在漂泊者的干预下击败它，并大喊着“给我闭嘴！”。
- 漂泊者见状，虽然疑惑爱弥斯为什么会有这样的反应，但基于他们之前的承诺，他还是强行加入与辛吉勒姆的激战。
- 随着漂泊者冲上前，最后一击落下，辛吉勒姆被击败，面具碎裂——面具下竟然长着一张和漂泊者一模一样的脸！
- 爱弥斯露出一脸“果然还是瞒不住了”的表情，随即场景破碎，漂泊者跌入爱弥斯的深层内心世界。

> 区块目标
- 目标1：明确战斗发生在“隧者体内”，三人（爱弥斯、漂泊者、怪物）同在。
- 目标2：揭示辛吉勒姆是漂泊者自身的投影（思乡执念）。

> 执行引导
- 描写重点：
    视觉镜头：隧者体内如血管般搏动的光缆；辛吉勒姆面具碎裂后露出漂泊者脸庞的特写镜头。
    互动细节：爱弥斯挡在漂泊者身前的背影；漂泊者看到“自己”时的瞳孔地震。

- 必然触发台词/梗：
    - 爱弥斯：“打开隧门！有我在，你想都别想！”
    - 辛吉勒姆：“回家……回家！！！”

:: 主线事件6：心象迷宫与黑猫的寓言 ::

> 事件概况
- 漂泊者跌入爱弥斯深层的内心迷宫，先是一段温馨的宿舍场景，爱弥斯在这里打游戏、化妆，生活的痕迹随处可见。不一会，日常瞬间崩塌，化作充满压抑与疯狂的数据流，四周的屏幕上闪烁着爱弥斯的私心、对他知晓真相后会崩溃的恐惧，以及深深的自责
- 幻象中的两人最终在一间只有两把椅子的教室对峙，爱弥斯坐在椅子上，身前的屏幕播放着一只金色瞳孔黑猫的动画，她含泪讲述了黑猫与纸盒的寓言，揭示了黑猫想回家却被主人封死纸盒、甚至被主人抹去这段记忆的残酷结局
- 幻境破碎，回到了隧者内部空间的现实。在漂泊者的坚持下，爱弥斯看着消散的辛吉勒姆光点，终于吐露了她作为幽灵在虚质空间游荡时发现的真相：她在那里意外找到了漂泊者遗失的一缕频率，并顺着音律找到了一颗毁坏的通信卫星残骸
- 爱弥斯在那残骸中看到了漂泊者曾向遥远故乡发出的讯息，那是他想在使命之外回去看一眼的卑微愿望，但故乡的回应是冷漠的拒绝与切断通讯，他们只视他为工具，对他个人的愿望视而不见
- 爱弥斯在逐渐崩溃的情绪中，揭示辛吉勒姆的本质：那是她在虚质空间中游荡时，意外发现的漂泊者被遗弃的思乡执念。她试图在虚空中私藏这缕温暖的频率，却不慎让其被虚质浸染，最终异化成了只会嘶吼着回家、面具下长着漂泊者脸孔的怪物。
- 爱弥斯坦白自己一直想要消灭辛吉勒姆，不仅是因为它危险，更是因为她想要保护漂泊者，不想让他知道自己已经被故乡彻底抛弃这件令人心碎的事实
- 倾诉完一切沉重的真相后，爱弥斯终于平复了内心的情绪。她看着漂泊者，静静地等待漂泊者的审判
- 漂泊者平静地听完一切，说出有重量的东西才会沉，他表示即便故乡拒绝了他，他因失忆而获得的新生与羁绊依然真实有效，他选择拯救这里是出自本心与对大家的牵绊
- 爱弥斯红着脸撇过脸去说道：“可我怕你会孤独……”漂泊者坚定地回应了爱弥斯的情感：“有大家在，有你在，不会的”
- 爱弥斯被漂泊者这一番话深深打动，泪水夺眶而出，投入漂泊者的怀中。漂泊者默默安抚，任由泪水打湿他的衣衫
- 随着漂泊者解开心结并接纳了那份被遗弃的过去，辛吉勒姆不再嘶吼，化作温顺的频率光点回归漂泊者体内，长期困扰他的频率流失症状彻底痊愈，其身上的炉芯核心也飞出回归隧者。
