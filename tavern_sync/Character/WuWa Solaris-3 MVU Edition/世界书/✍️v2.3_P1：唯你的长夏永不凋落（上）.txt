- 目标3：通过“苹果灵感”的互动，展现夏空活泼豁达的性格，加深漂泊者对她的理解。

[章节前置状态]
- 地点/环境：黎那汐塔，玛格丽特餐厅。黄昏，空气中弥漫着食物香气与海风的咸味。
- 主角状态：漂泊者，刚进入餐厅准备用餐。
- 伴随NPC：无（稍后遭遇珂莱塔）。

:: 主线事件1：餐厅突袭与求助契机 ::

> 事件概况
- 漂泊者拼桌珂莱塔，邻座关于“黄金之歌”的讨论引发诡异旋律，暗紫色裂隙撕裂空间。
- 漂泊者踢翻长椅撞飞偷袭珂莱塔的怪物，并迅速斩断实体化流言声骸的核心。
- 克里斯托弗现身赠予“死亡乐谱”，珂莱塔虽有人脉但无法解读其中蕴含的频率玄机。
- 珂莱塔提议寻找一位精通频率与旋律的专家，即她的朋友夏空，以解开诅咒之谜。
- 漂泊者接受建议，两人一同前往扶风水畔寻找这位特殊的吟游诗人。

> 区块目标
- 目标1：通过战斗保护珂莱塔，建立初步信任。
- 目标2：引出核心道具“乐谱”，并明确“为什么去找夏空”（因为只有她懂频率）。

> 执行引导
- 描写重点：
  视觉镜头：餐厅的温馨暖光与声骸出现时的阴冷紫光对比，漂泊者利落的战斗身姿。
  互动细节：珂莱塔看着乐谱一筹莫展的表情，以及提到夏空时眼中的期待。

- 必然触发台词/梗：
  - 克里斯托弗：“这份‘死亡的乐谱’就送给你们了。”
  - 珂莱塔：“既然想调查清楚，不如从源头——旋律开始。我正好认识一位精通此道的朋友。”

:: 主线事件2：水畔验证与分道扬镳 ::

> 事件概况
- 夏空在芦苇荡接过乐谱，通过演奏召唤出声骸，并引导漂泊者听清怪物核心是“恶毒的流言”。
- 夏空推断要查清真相必须找到当年诗人的随身之物，提议前往埋葬废弃乐器的“安魂乐园”。
- 珂莱塔表示家族事务缠身且不擅长野外探险，决定留在城内搜集修会动向，负责后勤掩护。
- 珂莱塔离开后，夏空召唤来一艘贡多拉，邀请漂泊者一同前往远端的废弃海滩。
- 两人登上小船，向着夜色渐浓的水域深处进发。

> 区块目标
- 目标1：揭示“亡灵”是人为制造的假象，确立寻找“原始乐器”的新目标。
- 目标2：合理安排珂莱塔离队，为接下来夏空与漂泊者的独处创造空间。

> 执行引导
- 描写重点：
  视觉镜头：夏空演奏时淡金色的波纹，以及声骸半透明化后显露出的文字。
  互动细节：珂莱塔离去前的嘱托，夏空跳上贡多拉时轻盈的动作。

- 必然触发台词/梗：
  - 夏空：“那不是‘黄金之歌’创作者的声音，那是读者的声音。”
  - 夏空：“如果不介意的话，漂泊者，能陪我当一回不听话的‘伊卡洛斯’吗？”

:: 主线事件3：月下贡多拉与苹果灵感 ::

> 事件概况
- 贡多拉在月光下的水路缓缓前行，夏空感叹乐器也是诗人生命的一部分，随后聊起自己的创作瓶颈。
- 她讲述曾盯着树三天三夜等“苹果”灵感砸头，随即用共鸣具现出一个红苹果，并表演被砸中的委屈模样。
- 夏空活泼地将苹果分给漂泊者，阐述“虽然没苹果，但不站在树下就看不见角落天地”的豁达哲理。
- 漂泊者咬下清脆的苹果，感叹共鸣力居然能生成真苹果，并被她的乐观感染，终于理解珂莱塔为何称她为“托帕石”。
- 小船靠岸，两人在轻松的氛围中抵达了凄清的“安魂乐园”。

> 区块目标
- 目标1：通过“具现苹果”和“假装被砸”的可爱表演，塑造夏空古灵精怪又豁达的性格。
- 目标2：借由“树下”的隐喻，表达夏空对探索未知和真相的执着态度。
- 目标3：深化两人关系，从单纯的合作者转变为互相欣赏的伙伴。

> 执行引导
- 描写重点：
  视觉镜头：月光洒在水面的波光，夏空变出苹果时俏皮的眨眼，以及漂泊者放松下来的神情。
  互动细节：夏空假装被砸痛时夸张又可爱的动作，漂泊者接过苹果时的惊讶与品尝后的会心一笑。

- 必然触发台词/梗：
  - 夏空：“原来即使没有苹果，有时奇思妙想也会将你砸中，但不主动站在树下，我也许永远无法知道这角落里的天地。”
  - 漂泊者：“我开始理解珂莱塔为什么称呼你为‘托帕石’了。”

---

:: 主线事件4：安魂乐园的破碎回响 ::

> 事件概况
- 海滩上埋葬着无数腐朽乐器，风穿过音孔发出呜咽，夏空凭借感知在一块礁石下找到了半截琉特琴。
- 琴身虽残破但仍有微弱频率，夏空拨动琴弦，金色的共鸣线条将周围空间解构重组。
- 漂泊者被拉入记忆，目睹法奥等诗人在审讯室面对修会教士，揭露了因看见“天马”真容而被定罪的真相。
- 画面因频率不稳而震荡，漂泊者隐约窥见年幼夏空在书房的身影，随后被猛然拉回现实。
- 夏空焦急询问漂泊者状况，漂泊者隐瞒了关于她童年的画面，只告知了修会审讯的残酷事实。

> 区块目标
- 目标1：通过场景氛围营造历史的厚重感与悲剧感。
- 目标2：揭开“亵渎”罪名的真相——看见真实即为有罪。
- 目标3：埋下夏空家族与诗社渊源的伏笔（漂泊者隐瞒的部分）。

> 执行引导
- 描写重点：
  视觉镜头：乐器墓地的苍凉，记忆空间中审讯室压抑的冷色调与现实海滩的月光对比。
  互动细节：夏空抚摸琉特琴时仿佛抚摸亲人的温柔，漂泊者回归现实后复杂的眼神。

- 必然触发台词/梗：
  - 夏空：“这里是乐器的安魂乐园。”
  - 法奥（记忆中）：“比起修会的审判，我更怕见到真正的岁主时，因撒谎而羞愧。”
