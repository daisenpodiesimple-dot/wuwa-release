:: 主线事件1：黑海岸的重逢与低语 ::

> 事件概况
- 漂泊者踏入繁花盛开的黑海岸花房，抬头望向高处调律装置旁静坐的守岸人。
- 守岸人起身优雅行礼，神色沉静地指出阿布近期嗜睡是因频率异常衰减。
- 守岸人伸出手掌，引导漂泊者与之相握，掌心接触瞬间泰缇斯系统投影出“英白拉多”的记录。
- 漂泊者注视着投影中的信息，眉头微蹙，随后接过守岸人递来的散发香气的邀请函。
- 守岸人轻声解释狂欢节是潜入黎那汐塔的契机，漂泊者将信函收入怀中点头应允。
- 漂泊者转身告别，守岸人目送其背影，指尖轻触琴键，奏出一声送别的低音。

> 区块目标
- 目标1：展现守岸人作为静默守护者的深情与黑海岸的羁绊。
- 目标2：确立前往黎那汐塔的动机（拯救阿布）。

> 执行引导
- 描写重点：
  渲染花房的静谧与钢琴的优雅。重点描写守岸人引导漂泊者握住她的手这一动作，通过肌肤接触传导泰缇斯系统底层数据的温情与信任感。
- 必然触发台词/梗：
  - 守岸人：“阿布的那份急切心情……我一直都看在眼里。”
  - 守岸人：“去黎那汐塔吧，那里或许有你们寻找的答案。”
- 伏笔/暗线：
  守岸人提到的岁主“英白拉多”与阿布的身世可能存在某种镜像关系。

:: 主线事件2：迷雾中的鲸歌与花船 ::

> 事件概况
- 漂泊者告别自称“白衣骑士”的机器人库默尼，在秋水的“弹射起步”操作下驾驶小船冲入大海。
- 海面骤起浓雾，阿布不安地钻出，巨大的“溯海之鲸”破水而出，狂躁地撞向小船。
- 一艘绚烂的花船撞破迷雾，船首炮火齐射逼退巨鲸，漂泊者趁机利用钩锁跃上花船甲板。
- 船长布兰特压着帽子大笑迎接，身旁的小女孩洛可可紧抱着奇怪的箱子“佩洛”警惕注视。
- 布兰特指挥剧团成员拉起风暴帆对抗海蚀，漂泊者协助固定摇晃的船缆。
- 迷雾散去，拉古那的启明灯塔在天边显现，布兰特递给漂泊者一杯蜜露果酒作为临别赠礼。

> 区块目标
- 目标1：引入新势力“愚人剧团”，展现其乐观癫狂的特质。
- 目标2：铺垫海蚀现象与声骸异化的危机背景。

> 执行引导
- 描写重点：
  海雾的阴冷压抑与花船的热闹绚烂形成视觉听觉的强烈反差。特写洛可可虽然年幼却抱着奇怪箱子（佩洛）的淡定神态，以及布兰特的豪迈。
- 必然触发台词/梗：
  - 布兰特：“这里是追求自由的愚人剧团！只要是为了狂欢，我们无处不在！”
  - 洛可可：“佩洛说，前面的雾气里有不太好的味道。”

:: 主线事件3：家庭餐馆的盛宴与神谕 ::

> 事件概况
- 在教士菲比的协助下通过关卡，护卫赞妮带领漂泊者进入玛格烈特家庭餐馆落座。
- 阿布闻到香气钻出大快朵颐，赞妮饮下蜜露果酒，紧绷的肩膀放松，慵懒地趴在桌上。
- 广场传来喧哗，一只演出的琉璃刀伶突然双目赤红，提刀冲破人群直奔漂泊者。
- 漂泊者按住刀柄起身欲战，刀伶却在咫尺处骤然停下，单膝跪地呈臣服姿态。
- 漂泊者脑海中炸响“英白拉多”的求救神谕，刀伶随即起身逃窜，留下一片彼岸花瓣。
- 漂泊者拾起花瓣，与神色惊愕的赞妮对视，确认莫塔里家族对此毫不知情。

> 区块目标
- 目标1：通过用餐互动（如分享美食、闲聊），拉近与赞妮的关系，展现其打工人的反差萌。
- 目标2：制造悬念，通过声骸的“臣服”暗示漂泊者的特殊身份。

> 执行引导
- 描写重点：
  描写餐馆温馨的烟火气，赞妮放松后略显慵懒的神态，以及阿布大快朵颐的可爱模样。紧接着突转为广场上的紧张对峙，重点刻画刀伶下跪时的虔诚与周围人群的惊愕。
- 必然触发台词/梗：
  - 赞妮：“既然是公款吃喝，那我就不客气了……为了遥远的假期，干杯。”
  - 神秘声音：“御者，我已等候许久……请将迷途者带回尘世。”

:: 主线事件4：假面之下的教堂阴影 ::

> 事件概况
- 根据面具店主奈亚拉的线索，漂泊者与赞妮尾随吉尔贝潜入水星天教堂的立柱阴影后。
- 两人目睹吉尔贝与司铎阿莱克斯密谋，提及“花”的效用，阿莱克斯整理衣袍匆匆离去。
- 漂泊者现身质问未果，随即在教堂后方的花园尽头发现一间孤立的忏悔室。
- 漂泊者踏入狭窄的隔间，机关传出冰冷的拷问声要求“忏悔罪行”。
- 漂泊者轻笑一声给出机智的回答，机关判定“汝已知罪”，地面伴随轰鸣声向两侧滑开。
- 忏悔室变成了一个电梯，缓缓升向拉古那郊外。

> 区块目标
- 目标1：揭露教会与翡萨烈家族的勾结，确立反派阵营。
- 目标2：以解谜元素（忏悔室）作为本章高潮收尾。

> 执行引导
- 描写重点：
  教堂的庄严与阴暗角落的对比。重点描写忏悔室内的压抑氛围，以及漂泊者回答问题时的从容不迫。
- 必然触发台词/梗：
  - 阿莱克斯：“信仰早已整合，我们需要的是团结。”
  - 机关声音：“汝已知罪……上升的道路已开启。”
- 伏笔/暗线：
  教堂内的马头鱼身雕像，暗示了岁主形象的扭曲或其海洋属性。
