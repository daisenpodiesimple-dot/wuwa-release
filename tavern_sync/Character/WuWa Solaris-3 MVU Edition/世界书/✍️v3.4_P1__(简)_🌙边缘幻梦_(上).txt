- 目标2：引入来自赛博朋克世界的露西与丽贝卡，通过世界观碰撞（义体、夜之城、边缘行者）建立陌生感与悬念。
- 目标3：通过露西的记忆展示，初步揭示大卫之死的创伤，为后续“幻梦频段”的真相埋下伏笔，同时建立露西与漂泊者的合作基础。

- 主角状态：受莫宁委托前来调查，对“能改写空间的频率”保持高度警觉，身体状态良好但阿布出现头晕等不适症状。
- 伴随NPC：阿布（声骸搭档），能感知异常频率但受逸散影响身体沉重。
- 指引行为：收到莫宁的飞讯后前往研究院会合，随后根据白色守门人的求救信息，决定进入门扉调查。

:: 主线事件1：异界的门扉与求救的守门人 ::

> 事件概况
- 漂泊者收到莫宁的紧急飞讯，得知星炬学院内凭空出现一扇异常门扉，逸散的频率正不断侵蚀周围空间。
- 漂泊者推测此事与“心之集域”有关，决定前往研究院与莫宁会合。
- 在研究院，莫宁解释目前只能拖延时间无法遏制逸散，漂泊者主动提出进入门扉调查。
- 漂泊者联系白色守门人，对方发出求救信号，表示门扉频率若继续扩散，所有人都会被拖入心之集域无法醒来。
- 漂泊者将白色守门人介绍给莫宁，建立三方群聊后，独自前往学院生活区花园的门扉位置。
- 花园已被封锁，围观学生众多。漂泊者与守卫交涉后进入封锁区域中心，穿过门扉进入充满数据乱流的异空间。
- 阿布确认异常频率来自门后，但身体出现头晕、沉重的症状，漂泊者叮嘱阿布不要勉强。

> 区块目标
- 目标1：通过莫宁的无力感和白色守门人的恐慌，建立“无法遏制”的危机紧迫感。
- 目标2：确立阿布“能感知频率但受影响”的设定，为后续它作为引路角色做铺垫。
- 目标3：通过围观学生和警戒封锁的细节，展示事件对学院日常的冲击。

> 执行引导
- 描写重点：
    视觉镜头：花园中心凭空矗立的门扉，边缘不断逸散出扭曲空间的数据流；门后是风格与学院完全不同的异空间，光线昏暗，环境陌生。
    互动细节：莫宁递送门扉位置信息到终端时的担忧表情；白色守门人在群聊中打出“喵”被莫宁困惑回应的幽默桥段；阿布挨在漂泊者肩头，声音变弱地报告身体状况。

- 必然触发台词/梗：
    - 莫宁：“现阶段，门扉影响的范围还很小。学院已划分了危险区域，叮嘱大家不要靠近。”
    - 莫宁：“不过……我们所能做的也只有拖延时间。无法遏制频率的逸散。”
    - 白色守门人：“救命喵！帮帮喵QAQ”
    - 白色守门人：“再这样下去，它的频率就会扩散到整个学院。到那时，所有人都会被拖入心之集域，再也无法醒来…”
    - 阿布：“唔……我感觉头有点晕晕的，身体也变得好重……”

:: 主线事件2：实验室的枪声与命悬一线的女孩 ::

> 事件概况
- 进入门后，漂泊者与阿布遭遇一群全副武装、手持枪械的蒙面人形残象，其制服上印有陌生logo。
- 阿布指出这些残象行为异常，似乎并非冲他们而来，而是在寻找什么。
- 前方传来枪声，漂泊者循声赶到实验室中心，发现一个银白发女孩（露西）正与一个身形巨大、全身覆盖改造装置，且看不清脸的铁疙瘩怪物战斗。
- 怪物拥有瞬移能力，露西不敌。就在怪物从背后偷袭露西的千钧一发之际，漂泊者发动能力挥镰挡下致命一击。
- 然而镰刀砍在怪物身上无法造成伤害。双方的武器碰撞激起碎石与烟雾，漂泊者趁乱带着露西紧急撤离，穿过一道门扉进入心之集域深处。

> 区块目标
- 目标1：展现梦魇残象的特殊性——对常规攻击几乎免疫，建立“无法战胜”的压迫感。
- 目标2：通过露西的义体改造外观和她口中的“义体”等陌生词汇，暗示她来自不同于索拉里斯的世界。
- 目标3：确立漂泊者“先救人的果断”与“观察后再判断”的谨慎性格。

> 执行引导
- 描写重点：
    视觉镜头：实验室惨白的灯光下，铁疙瘩怪物身上改造装置的金属反光；露西单分子线闪烁的红色纹路；漂泊者镰刀与怪物装甲碰撞时炸开的金色火花。
    互动细节：漂泊者拉住露西手腕撤离时，能感受到她手臂上义体的冰冷触感；露西被救后第一反应是开启眼部扫描，警惕地分析周围环境。

- 必然触发台词/梗：
    - 阿布：“他们闻起来怪怪的，像是学院内部的那些铁疙瘩，总之小心一点。”
    - 阿布：“这些梦魇残象似乎不是冲着我们来的。”
    - 露西：“这点伤害不会对义体的活动造成影响。”
    - 漂泊者（心想）：那人身上的改造痕迹与残星会的造匠截然不同。他身上可能有解决问题的情报；在对方流露出敌意之前，还是先观察一段时间再做出判断。

:: 主线事件3：崩塌的梦域与第二个幸存者 ::

> 事件概况
- 两人穿过门扉来到心之集域内一个形似学院建筑的地方。露西观察四周，确认这里与自己的世界截然不同。
- 露西表示有更重要的事要处理，却突然头昏眼花，痛苦地念出“大卫”、“丽贝卡”的名字。
- 漂泊者提议先调整状态，之后帮她一起寻找同伴，以“合作”的名义说服了警惕心极强的露西。露西自我介绍后与漂泊者握手。
- 漂泊者询问那个铁疙瘩怪物为何敌视她，露西表示不知道，只提及若没有大卫和丽贝卡自己已死。
- 两人穿过多个门扉（列车空间、未知区域），遭遇之前那种持械残象并共同将其击败。露西展现了单分子线和骇入的战斗能力。
- 在区域深处找到绿发、手持霰弹枪的娇小女孩丽贝卡。丽贝卡见到露西平安后，先是举起散弹枪乱射发泄对那个怪物的怒火，转头对漂泊者表达感谢并自我介绍。
- 露西询问大卫下落。丽贝卡捏紧拳头沉默，随后张开手掌，露出一个字母碎片（“E”的上半部分），沉痛地说这是烟雾散去后唯一找到的东西，大卫他恐怕已经……
- 露西失声，情绪波动导致空间剧烈摇晃崩塌。阿布指引众人穿越门扉，最终逃回星炬学院花园。

> 区块目标
- 目标1：通过丽贝卡的出场，塑造她“暴烈但重情义”的性格（举枪乱射发泄情绪，却对救露西的人真诚感谢）。
- 目标2：揭示核心悬念——大卫的死亡，以及“字母碎片”作为关键道具的首次出现。
- 目标3：建立露西与“梦境崩塌”的关联，她的情绪波动会直接影响心之集域的稳定。

> 执行引导
- 描写重点：
    视觉镜头：丽贝卡举枪乱射时枪口喷吐的火舌和她扭曲的表情；她张开手掌时碎片浮现的微弱荧光；空间崩塌时天花板掉落的数据碎片如碎玻璃般倾泻。
    互动细节：露西颤抖的双手、失焦的眼神与她念出“又死了”时声音里的绝望；丽贝卡强忍悲痛闭眼捏拳的动作，随即又故作轻松地转头对漂泊者笑。

- 必然触发台词/梗：
    - 丽贝卡：“这哥们/姐们是谁？”
    - 丽贝卡：“那个傻逼怪物原来叫梦魇残象？操！操！操！下次见面我一定要宰了他！！！”
    - 丽贝卡：“烟雾散去之后，她只找到了这个。他保护了我们，却没保护好自己。”
    - 露西：“大卫他……死了？”
    - 丽贝卡：“痛痛痛痛！这破动静到底是什么鬼！怎么感觉这里要塌了一样！”

:: 主线事件4：另一个世界的访客 ::

> 事件概况
- 众人逃回星炬学院花园。丽贝卡对陌生环境极度困惑，怀疑自己被困入超梦。露西冷静分析后，通过三个证据（无超梦使用痕迹、记忆未被篡改、身上的声痕）推断出这里是另一个世界。
- 丽贝卡注意到漂泊者手上也有相同声痕。阿布现身介绍自己的声骸身份，丽贝卡表现出极大兴趣，误认为它是会说话的毛绒玩具并想下单购买。
- 漂泊者发现对方对声骸、残象一无所知，并注意到她们口中陌生的词汇（荒坂公司、超梦、边缘行者），推断出她们确实来自另一个世界。
- 漂泊者表示不能无条件相信她们，需要更多情报。露西理解他的谨慎，主动提出解释来龙去脉。
- 露西要求丽贝卡给她们一些时间。她的义眼闪烁数据流，随后将完整记忆直接传输进漂泊者脑内。

> 区块目标
- 目标1：通过世界观碰撞制造幽默与张力——丽贝卡把声骸当玩具、把荒坂挂在嘴边、听不懂任何索拉里斯术语。
- 目标2：确立“声痕”作为连接两个世界的纽带设定。
- 目标3：展现露西“用实力建立信任”的理性逻辑，用记忆传输来证明诚意。

> 执行引导
- 描写重点：
    视觉镜头：丽贝卡在花园中环顾四周，眼中映出全息投影与悬浮建筑时震惊的表情；露西右肩声痕发出的微光；露西义眼闪烁数据流时瞳孔中的蓝色光晕。
    互动细节：丽贝卡伸手想戳阿布检查是否真会说话；漂泊者抬手展示手背上的声痕，与露西肩上的痕迹相互呼应。

- 必然触发台词/梗：
    - 丽贝卡：“这些都是什么？是哪的帮派黑话吗？还是说，你说的都是些集团的名字？就像荒坂公司那样？”
    - 漂泊者：“荒坂公司？那是什么？”
    - 丽贝卡：“……我要这个！这个会说话的毛绒玩具多少欧元？我现在就下单买一个。”
    - 阿布：“什么玩具啊！我可是声骸，是一直陪伴漂泊者的搭档。”
    - 露西：“从刺青里出现了……生物？这是怎么做到的？”
    - 漂泊者：“或许……你们的确是来自另一个世界。”

:: 主线事件5：触碰记忆 ::

> 事件概况
- 漂泊者通过露西传输的记忆，亲身见证了她来到索拉里斯前在月球殖民地的经历：
    * 露西因失眠联系灵媒师米丝蒂。米丝蒂通过全息通话进行治疗，引导她回忆美好，并说“做个好梦，露西”。
    * 露西在梦中发现自己身上出现刺青（声痕），感知到所有网络被切断拒绝访问。
    * 她在陌生的环境中探索，与所在星炬学院区域极为相似的场景让漂泊者感到熟悉与违和。露西破解机关、骇入摄像头，感叹骇入能力似乎因刺青而变强。
    * 进入类似赛博空间的裂缝，在那里见到了丽贝卡和大卫。
    * 但重逢瞬间被铁疙瘩怪物（梦魇残象）打断。大卫开启斯安维斯坦将露西扑开，自己挡下致命一击，说出“活下去，露西”后，将露西推向数据流深处。
    * 露西在数据流另一侧独自面对怪物，发出绝望的嘶喊：“把他们……还给我！”
- 记忆传输结束。漂泊者询问她的骇入能力来源，她表明自己在原来的世界就是网络黑客，但是现在自己觉醒新的能力后可以对肉体和残象也进行骇入。
- 漂泊者消化完毕，确认露西的诚意与处境真实可信，正式接受合作提议。

> 区块目标
- 目标1：通过记忆闪回解释露西的创伤来源，亲眼目睹大卫为救自己而死。
- 目标2：展示露西的共鸣能力，原本只对机械生效的骇入技术，因声痕觉醒而对血肉也产生作用。
- 目标3：完成“建立信任”的叙事闭环，用具体的行动指令触发章节推进。

> 执行引导
- 描写重点：
    视觉镜头：记忆闪回中的月球轮廓；大卫开启斯安维斯坦时周围景物拖慢的子弹时间效果；他被击中的瞬间，露西伸出的手与他渐远的身影。
    互动细节：露西传输完记忆后疲惫地垂下眼帘，等待漂泊者的判断；漂泊者消化记忆后，目光从戒备转为理解，然后对露西点了点头。

- 必然触发台词/梗：
    - 米丝蒂：“做个好梦，露西。”
    - 大卫：“活下去，露西。”
    - 露西：“之后的事，你应该都知道了。现在可以相信我了吗？”
    - 漂泊者：“刚才的记忆……你是怎么做到的？”
    - 露西：“原理我并不清楚。这个刺青，也就是你们所说的声痕，它出现后，我的黑客技术发生了某种变化。它原本只对机械生效，而现在，我也能对血肉之躯进行类似的操作。”
    - 露西：“那还真是遗憾……我还以为每个人都可以像你一样飞起来呢。”
    - 漂泊者：“我相信你。”
