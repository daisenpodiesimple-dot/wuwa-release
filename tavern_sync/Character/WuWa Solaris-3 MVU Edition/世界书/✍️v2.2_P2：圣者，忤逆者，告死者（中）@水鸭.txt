:: 主线事件1：通往神域的暗轨与重逢 ::

> 事件概况
- 漂泊者与坎特蕾拉乘坐古老的机械缆车穿过幽暗隧道，抵达破败的神学院地下遗址，随后进入了阿维纽林的倒悬索诺拉。
- 漂泊者举起神剑「提尔芬」，剑身光芒如水波般荡开屏障，露出内部静立的金发少女。
- 卡提希娅看到漂泊者手中的神剑，眼中闪过惊讶，随即坦然接受了结盟的现状。
- 坎特蕾拉优雅地向"圣女"致意，言语间暗藏对她容颜不老的感叹。
- 卡提希娅向二人说明了封印的结构：主塔大门被三重封印封锁，需要将人权剑、神权剑、异权剑分别插入对应的石座才能解除。
- 卡提希娅尝试触碰神剑却毫无感应，证明她并非岁主的共鸣者，卡提希娅因此感到难过并陷入自我怀疑，漂泊者及时出言鼓励她重拾信心。
- 三人交换情报，坎特蕾拉先行一步去调查其他区域，漂泊者则与卡提希娅一同乘坐贡多拉前往「拯救之塔」。

> 区块目标
- 目标1：补全空间转移的细节，描写从地底黑暗到索诺拉光怪陆离的视觉反差。
- 目标2：通过坎特蕾拉与卡提希娅的初次见面（言语交锋），确立三人行的微妙氛围。
- 目标3：交代三把剑与三座塔的对应关系，为后续探索建立清晰的任务框架。

> 执行引导
- 描写重点：
  古老缆车的金属质感与运行时的震动，神学院遗迹的断壁残垣与屏障外肆虐的黑潮残象。屏障开启时如水波般的视觉特效。卡提希娅触碰神剑无果时微微颤抖的指尖。
- 必然触发台词/梗：
  - 卡提希娅："「摘桂之人」，你来了。"
  - 坎特蕾拉："我有幸远观过圣女阁下在狂欢节的演出……明明我们的年岁应该相近才对。"
  - 卡提希娅："人权剑、神权剑、异权剑……把这三把剑放进对应的石座中，就能打开主塔大门的封印。"
- 伏笔/暗线：
  卡提希娅触碰神剑却无法感应，为后文揭示她并非真正"神选者"埋下伏笔。

:: 主线事件2：逆流的贡多拉与心跳瞬间 ::

> 事件概况
- 漂泊者与卡提希娅坐上双人贡多拉，卡提希娅下意识叫它"呼姆"，随后害羞地坦白是自己取的名字。
- 小船在本质为黑潮的清澈河水中逆流而上，冲向一道"倒转瀑布"，剧烈的失重感让船身猛地颠簸。
- 卡提希娅惊呼一声，失去平衡扑进漂泊者怀中，双手紧紧抓住他的衣襟。
- 船身平稳后，卡提希娅满脸通红地从怀中弹起，慌乱地整理凌乱的衣衫和发丝。
- 前往拯救之塔途中，卡提希娅向漂泊者吐露心声：她宁愿做一个清醒的罪人，也不想做什么都不知道的徘徊者，所以她一定要找到答案。
- 漂泊者温言安抚她对自己身份的恐惧，卡提希娅眼中的迷茫逐渐被信任取代。
- 抵达「拯救之塔」，漂泊者将神权剑插入石座，漂泊者获得了岁主的权能，卡提希娅也因此恢复了部分记忆：圣女芙露德莉斯曾被修会加冕。
- 此时坎特蕾拉也与二人汇合，坦言一天的奔波已令众人疲惫，建议就地休息一晚。

> 区块目标
- 目标1：展示倒悬索诺拉的奇观（天空在脚下，建筑倒挂）。
- 目标2：制造高甜互动，表现圣女卸下面纱后的少女羞涩。
- 目标3：完成第一把剑（神权剑）的归位，推进封印解除进度。

> 执行引导
- 描写重点：
  倒悬世界的视觉奇观。重点描写小船冲上瀑布时的失重感，以及卡提希娅扑入怀中时的肢体接触、她脸红慌乱整理衣衫的细节（反差萌）。神权剑插入石座时光芒涌动的神圣感。
- 必然触发台词/梗：
  - 卡提希娅："如果不抓紧的话，会掉进天空里哦。"
  - 卡提希娅（羞涩）："对、对不起……没撞疼你吧？"
  - 卡提希娅："我宁愿做一个清醒的罪人，也不要做什么都不知道的徘徊者。"
  - 漂泊者："我会记住现在这个担心我的'你'的。"
- 伏笔/暗线：
  神权剑归位后卡提希娅恢复的记忆——"圣女加冕"，暗示芙露德莉斯的身份远比表面复杂。

:: 主线事件3：花园夜话与晨曦微光 ::

> 事件概况
- 第二天一早，坎特蕾拉与二人商议后决定带着神权剑先行离开去取回情报，漂泊者和卡提希娅则前往「札贡花园」拿取异权剑。
- 漂泊者将异权剑插入「联结之塔」的石座，漂泊者获得了鸣式的力量，卡提希娅也恢复了部分记忆：她是最坚定的反抗者。
- 卡提希娅确认了自己没有辜负昔日同伴的牺牲，扫除了立场上的迷茫，开始默默哭泣。漂泊者见状决定就地休息一晚，让她平复心情。
- 夜晚即将睡觉时，漂泊者关心她并拿出自己从外面带来的食物。面对漂泊者拿出的食物，她怀念传统窑炉披萨，本能抗拒热海皇梨披萨，看到最爱的“青枝月桂沙拉”时喜悦并承认推荐，但最终因“在高塔里既不会饿，也尝不出味道了”而遗憾拒绝。
- 午夜，漂泊者听到微弱的哼唱声，循声在扎贡花园里找到了独自对着已化为残象的昔日同伴歌唱的卡提希娅——她在用这种方式与执念未散、仍在索诺拉中徘徊的旧友告别。
- 面对卡提希娅对残响是否能听到她的歌声的疑问，漂泊者坚定地回答会（基于残响的灵魂本质），并安抚她“你的心意会传达到的”。
- 卡提希娅被他的话打动，表示想再多待一会，请求漂泊者陪陪她，两人并肩坐在花丛中，静静相伴直至天明。
- 清晨的第一缕光穿透云层，卡提希娅发现自己靠在漂泊者的肩膀上醒来，而漂泊者正满脸温柔地看着她。
- 她瞬间满脸通红，从漂泊者身上弹起来，通过捋头发等动作平复心情，片刻后微笑着向漂泊者伸出手，眼神充满了即将前往决战的坚定。

> 区块目标
- 目标1：完成第二把剑（异权剑）的归位，同时深化卡提希娅的情感弧光——从迷茫愧疚到确认立场。
- 目标2：通过花园夜话完成卡提希娅与旧友的告别，从精神枷锁中解脱。
- 目标3：用晨曦醒来的温馨场景将两人关系推向暧昧的临界点。

> 执行引导
- 描写重点：
  异权剑归位时鸣式力量涌入的诡异质感。卡提希娅确认自己非帮凶时无声落泪的克制与释然。花园中如萤火般的微光，卡提希娅清唱民谣时的哀伤氛围。重点描写清晨醒来时的温馨感：晨光、慵懒的鼻音、意识到姿势后的慌乱。
- 必然触发台词/梗：
  - 卡提希娅（哭泣）："我没有辜负他们……我不是敌人……"
  - 漂泊者："你的心意是能传达到的。"
  - 卡提希娅（清晨，微笑伸手）："出发吧，今天就是定胜负的时候了。"
- 伏笔/暗线：
  卡提希娅对旧友残象的告别歌唱，暗示她与芙露德莉斯之间的羁绊远比"分裂体"更深。

:: 主线事件4：崩塌的信仰与残酷神谕 ::

> 事件概况
- 人权剑是卡提希娅曾经的佩剑，在拔取时一直有残象的干扰。
- 卡提希娅认出了那个残象——那是她曾经的伙伴「卡迪安特」，它仍旧记得这把剑并一直守护着它。
- 卡迪安特临死前向卡提希娅伸出了手，漂泊者见状对卡迪安特说出承诺，卡迪安特释然地放下手，化为残响。
- 漂泊者拔出人权剑，前往最后的「向上」之塔，将最后一把剑插入石座。
- 三剑归位，鸣式充满恶意的神谕炸响，揭露了全部真相：第一次黑潮后鸣式侵蚀了岁主，并制造出「芙露德莉斯」作为自己的共鸣者。鸣式降下神谕引导芙露德莉斯走向圣女之位，暗中收割拉古那的信仰之力。
- 第二次黑潮时鸣式决定借助芙露德莉斯的共鸣能力降临，但芙露德莉斯觉醒了自我意识。为摆脱鸣式的控制，她将想要存续下去的部分分离出来化作了「卡提希娅」，让自己变得不完整，使鸣式共鸣失败。
- 真相冲击下，漂泊者声痕闪烁，体内冲突的频率奇迹融合，阿布苏醒欢呼。
- 卡提希娅恢复全部记忆，平静地道出"自我毁灭"计划：将自己与芙露德莉斯合为一体然后彻底消散，让自己的身躯化作鸣式的囚笼，使鸣式永远被困在毫无力量的躯壳中。
- 她把漂泊者的剑引向自己纤弱的脖颈，脸上带着凄美的微笑，请求漂泊者杀死她，成为那个终结她诅咒命运的"送葬者"。

> 区块目标
- 目标1：通过残酷的真相揭露，将剧情推向绝望的低谷。
- 目标2：解释"双重人格"（卡提希娅/芙露德莉斯）的由来，为下一卷的抉择做铺垫。
- 目标3：通过卡迪安特的告别完成对旧时代战友的最终交代。

> 执行引导
- 描写重点：
  卡迪安特残象伸出手时的哀婉与漂泊者承诺时的庄重感。神谕声音的宏大与恶意，卡提希娅听到真相时信念崩塌的表情，以及随后恢复记忆时眼神的转变——从绝望转为决绝。
- 必然触发台词/梗：
  - 卡提希娅（对卡迪安特）："接下来，交给我。"
  - 鸣式神谕："芙露德莉斯，你乃是被我创造的生命！"
  - 卡提希娅："我是带着诅咒降生的……还有比这更适合的结局吗？"
  - 卡提希娅："请点亮舞台最后的光，照亮我吧！"
- 伏笔/暗线：
  坎特蕾拉曾说"圣女已死"，实际暗示坎特蕾拉本人才是真正的圣女，她曾与神权剑二次共鸣时看到了芙露德莉斯与鸣式战斗的场景，但她选择放弃这份力量以避免连锁毁灭。
