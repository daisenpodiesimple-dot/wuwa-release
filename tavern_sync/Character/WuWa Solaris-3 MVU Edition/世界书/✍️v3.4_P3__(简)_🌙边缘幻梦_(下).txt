:: 主线事件1：反向解析与留守的决断 ::

> 事件概况
- 白色守门人告知露西，返回原世界的门扉位于心之集域的最深处，同时它表示会尽力准备一种紧急逃生手段以防万一。
- 为了在数据乱流中找到剩余的两块幻梦频段，漂泊者提出利用频段与梦魇残象频率截然相反的特性进行反向解析，莫宁立刻着手办理。
- 莫宁提出学院的常规武器无法抵御残象的入侵，但丽贝卡和露西的武器频率可以。她请求丽贝卡留下，配合研究院提取武器频率以武装学院师生。
- 丽贝卡爽快答应留下善后，并表示大卫醒来后最想看到的是露西，自己并不着急。
- 露西将自己的配枪留给丽贝卡防身。两人互相关照后，漂泊者与露西先行一步跨入通往心之集域深处的门扉。

> 区块目标
- 目标1：合理解释队伍拆分的原因，即武器解析需求与寻找频段的紧迫性。
- 目标2：埋下白色守门人最终逃生手段的伏笔，为后续的列车救援做铺垫。
- 目标3：展现丽贝卡粗中有细的体贴，她主动留下是为了成全露西与大卫的重逢。

> 执行引导
- 描写重点：
    视觉镜头：莫宁操作全息面板时快速流动的数据流；露西将配枪递给丽贝卡时两人指尖交错的特写；门扉在漂泊者和露西踏入后泛起的蓝色时空涟漪。
    互动细节：白色守门人提到逃生手段时心虚地舔了舔爪子；丽贝卡双手抱在脑后故作轻松地用下巴指了指门扉，催促露西快走。

- 必然触发台词/梗：
    - 白色守门人：“虽然没办法直接帮助你们，但我会尽最大的努力准备好逃生手段。”
    - 丽贝卡：“武器给你们研究？行啊，没问题。本来就是我们引起的麻烦，也该我们负责为其善后。”
    - 丽贝卡：“那个笨蛋更希望醒来看到的人，是你，露西。”
    - 露西：“我的枪先借你防身吧。如果梦魇残象入侵学院，可以用它暂作抵抗。”

:: 主线事件2：【双线并行】街头的涂鸦与公寓的投影 ::

> 事件概况
本事件采取双线交替并行的叙事方式，请根据主角所在位置挑选进行演绎。

[线索A：现实学院，丽贝卡的涂鸦与回忆]
- 丽贝卡在闲逛时遇到正对着墙面涂鸦发愁的琳奈与西格莉卡。这面墙正是丽贝卡之前喷涂大卫“ER”标志的地方，此刻墙体出现了数据化裂缝。
- 丽贝卡承认那是自己的杰作，并借机向两位女孩讲述了大卫以及边缘行者（佣兵）们的过去。
- 琳奈被故事打动，也在心里默默共情了她过去的雇佣兵生活，拿出多个喷罐邀请丽贝卡一起涂鸦。三人通宵创作，在墙上分别留下了黑猫、八哥犬和日灵的涂鸦图案。
- 大批公司狗残象突然从数据裂缝涌出。莫宁及时发来通讯确认已经远程适配完成了众人的武器频率，丽贝卡带领换上新武器的女孩们成功击退敌人。
- 战斗结束后，丽贝卡感叹两位女孩比看起来能打多了，涂鸦中心浮现出一块完整的金色频段字母R。西格莉卡推测是大卫感受到了思念才让碎片降临。
- 丽贝卡收起碎片，决定立刻去追赶露西。

[线索B：梦境夜城，露西的旧日残影]
- 漂泊者与露西进入梦域，发现环境变成了夜之城日本街的樱花集市，随处可见知名品牌广告与樱花树全息投影。
- 两人在阿布的感知协助下寻找路线并骇入防火墙。途中他们看到了学院旧日科考员的频率残影，这些关于虚质磁暴的讨论与露西对夜之城的记忆相互交织。
- 他们穿过了另一扇门，来到了大卫与露西曾经共同生活的豪华公寓。房间里保留着大卫不爱喝却为露西买的碳酸饮料，以及电视里循环播放的月球旅行广告。
- 露西走到露天阳台的泳池旁俯瞰夜景，向漂泊者吐露心声，指出在夜之城只有死亡才能成为传奇的残酷法则。
- 强烈的思念让阳台外具象化出了大卫的全息投影。露西感慨大卫死后，没有他的月球太过安静。平复情绪后，阿布感知到最后一块频段就在前方，两人继续迈向下一个门扉。

> 区块目标
- 目标1：通过涂鸦跨越世界观的隔阂，让丽贝卡在索拉里斯找到情感共鸣，并通过战斗获取第二块碎片。
- 目标2：通过回到家的场景，剖开露西对大卫深沉的爱与无尽的孤寂，将情绪铺垫至极点。
- 目标3：以平行的蒙太奇手法，对比丽贝卡的热血喧闹与露西的凄冷追忆，丰富叙事节奏。

> 执行引导
- 描写重点：
    [线索A] 视觉镜头：墙上鲜艳的八哥犬涂鸦与漆黑压抑的数据裂缝；换上新频率子弹后枪械射出的绚烂流光；悬浮在空中的金色字母R。
    [线索B] 视觉镜头：赛博朋克风的霓虹招牌与满天飞舞的全息樱花；公寓桌上积灰的碳酸饮料罐；大卫投影出现时透明的蓝色全息网格与露西落寞的侧脸。

- 必然触发台词/梗：
    - 丽贝卡：“即使加装了那些花里胡哨的军用级义体，还是没能阻止这一切的发生……我猜，他不想再体会失去重要之人的感觉了吧。”
    - 西格莉卡：“会不会……是大卫感受到有人在思念他，所以才出现在了这里？”
    - 露西：“在夜之城留名的方式从来不是如何活着，而是如何死去。”
    - 露西：“你知道吗，大卫……没有你的月球，太过安静了。”

:: 主线事件3：生死时速与坠入黑梦 ::

> 事件概况
- 漂泊者与露西穿过楼顶门扉，来到了梦境中夜之城的高速公路上。
- 梦魇亚当重锤突然发动伏击。千钧一发之际，丽贝卡骑着一辆星炬学院的科考摩托引发巨大爆炸赶来救援。
- 丽贝卡将借来的枪还给露西，并带来了另外两辆摩托的频率。三人骑上摩托，在夜之城的公路上展开躲避重锤导弹的生死追逐战。
- 重锤紧追不舍并炸断了桥梁，漂泊者带领队伍找到新通道，发现了通往下一个集域的门扉。然而门扉正在缓缓关闭。
- 丽贝卡和漂泊者全速冲刺成功滑入门内，但露西被重锤引发的爆炸气浪阻隔，被迫留在门外。
- 落单的露西不敌梦魇重锤的火力碾压，脱力倒下。
- 露西的意识坠入无尽的黑梦。在黑梦里，她不断听到过去同伴惨死的呼救声和指责她害死队伍的幻听。在濒临崩溃时，她听到了大卫在月球等她的声音。

> 区块目标
- 目标1：打造极具动作电影感的高速追逐戏，将赛博朋克的飙车元素与索拉里斯的能力相结合。
- 目标2：制造极其强烈的危机断点，即队友掉队与面临死亡处决，把露西推入心理防线的最低谷。
- 目标3：通过黑梦展现重度抑郁与幸存者内疚的折磨，并用大卫的呼唤作为最后的救赎锚点。

> 执行引导
- 描写重点：
    视觉镜头：丽贝卡连人带车砸下引发的火球；导弹擦过漂泊者脸颊引爆身侧汽车的慢动作；露西倒地后重锤巨大枪口的压迫感；黑梦中混乱交错的猩红字幕。
    互动细节：追逐战中丽贝卡单手控车向后开火的狂野姿态；门扉关闭瞬间漂泊者回头伸出手却抓空的绝望；露西在黑梦中痛苦捂住耳朵身体蜷缩成一团。

- 必然触发台词/梗：
    - 丽贝卡：“组派对怎么能不喊我呢！枪还给你。怎么样？这从天而降的救场是不是很帅！”
    - 梦魇亚当重锤：“可悲！虫子终归还是虫子！”
    - 梦魇亚当重锤：“承认吧，他们回不来了。与梦魇同化吧。我会解决你的痛楚！”
    - 过去的声音：“你的队友都是因你而死。如果没有你，大卫和丽贝卡也不会死在那一晚。”
    - 大卫的声音：“露西……我在██等你。”

:: 主线事件4：梦境列车与直面残酷的坦白 ::

> 事件概况
- 露西从黑梦中惊醒，发现自己躺在一辆疾驰的浮空列车车厢内，漂泊者、丽贝卡以及气喘吁吁的白色守门人都在身边。
- 原来这辆列车就是白色守门人准备的紧急逃生手段。列车冲破空间壁垒接住了露西，漂泊者趁机控制住残象将其救下。
- 看到依然活力四射的丽贝卡，露西的精神防线彻底崩溃。她不再自欺欺人，流着泪向大家坦白了现实中大卫和丽贝卡早已死于重锤之手的残酷真相，试图以此惩罚自己。
- 丽贝卡平静地表示自己早在一开始就知道了这个事实。她坦然面对死亡，并指出她们之所以能出现在这里，是因为露西还活着且记着他们的故事。
- 丽贝卡指出这终究是梦，梦醒之后露西必须去直面孤独的现实。漂泊者也上前承诺，至少这一次他们会陪她一起面对梦魇。
- 众人的决心让露西重新振作，列车也随之缓缓减速，停靠在心之集域的最深处。

> 区块目标
- 目标1：解开白色守门人逃生手段的伏笔，用猫开列车的荒诞感略微缓解前序的压抑氛围。
- 目标2：完成全剧最高潮的文戏，露西和盘托出大卫和丽贝卡死亡的真相。
- 目标3：展现丽贝卡作为边缘行者极其坦荡清醒的豁达人格，完成对露西的心灵救赎。

> 执行引导
- 描写重点：
    视觉镜头：列车窗外飞速倒退的扭曲数据流；白色守门人揉着泛红猫爪的可怜模样；露西浑身颤抖眼泪夺眶而出的破碎感；丽贝卡说出真相时释然的笑容。
    互动细节：丽贝卡想要安慰露西却停在半空的手，最终化为拍在露西肩膀上坚定的力量；漂泊者站在露西身侧用坚定的眼神给予无声的支持。

- 必然触发台词/梗：
    - 白色守门人：“你知道用猫爪开列车是件多么困难的事情吗？……现在爪子都还在疼呢。”
    - 露西：“现实中的大卫已经死了。你也一样，丽贝卡，你们都在那个夜晚死在了亚当·重锤的手里。”
    - 露西：“第二天入梦后我依然会回到夜之城，再一次目睹梦魇的诞生。什么都……无法改变……”
    - 丽贝卡：“听好了，露西，我和大卫的死亡是无法改变的事实。我们之所以会出现在这里是因为你还记着我们的故事……因为……你还活着。”
    - 漂泊者：“哪怕梦醒之后，你依然会是孤身一人。但至少这一次，我们会和你一起面对。”

:: 主线事件5：登月决战与梦幻的消散 ::

> 事件概况
- 众人走下列车，发现心之集域的最深处正是露西最渴望到达的地方：月球。最后一块幻梦频段（字母“E”的下半部分）悬浮在前方平台上。
- 梦魇亚当重锤再次降临阻拦众人。漂泊者与丽贝卡负责在前方掩护，为露西争取抢夺频段的时间。
- 重锤企图开启斯安维斯坦进入时间暂停状态，但漂泊者动用衍射属性的时序之能，在停滞的时间中撕开裂口，成功破解了重锤的杀招。
- 露西趁机冲上平台，将所有频段拼合。完整的大卫标志在月球上空爆发出耀眼光芒，大卫的身影随之具象化。
- 获得幻梦力量加持的众人，攻击终于瓦解了重锤的数据乱流，四人联手将梦魇亚当重锤彻底消灭。
- 战后，大卫微笑着与漂泊者和丽贝卡寒暄，露西则站在远处低头沉默不语。
- 丽贝卡将大卫推向露西身边，自己则转身向漂泊者发出加入边缘行者的玩笑邀请，随后知趣地留给两人独处空间。
- 大卫前去向露西打招呼，露西终于不再忍耐，冲上前流着眼泪，飞扑到大卫的怀里。
- 露西与大卫坐在陨石坑边遥望地球，露西抬手想要触碰他。大卫温柔地和她十指相扣，表示自己只是梦，现在梦该醒了，随后他的身体开始分崩离析。
- 露西固执地跨坐在他的腿上，手捧着他的脸，低头想要吻他，但大卫只是用脸蹭了蹭她的手，在被露西吻到的前一刻，消散在宇宙中，只留下一句感谢。
- 丽贝卡见状，上前主动抱住露西，在露西怀中消散并做下最后的道别。


> 区块目标
- 目标1：将最终战场设定在月球，完美契合边缘行者故事的终极浪漫与遗憾。
- 目标2：展现漂泊者破解斯安维斯坦的战斗能力，并促成大卫的复活与反杀。
- 目标3：完成大卫消散、丽贝卡释然拥抱以及与星炬学院告别的三段式离别，升华剧情的情感余韵。

> 执行引导
- 描写重点：
    视觉镜头：深邃宇宙下庞大的地球与荒凉的月壤；漂泊者发动时序之能时空间中凝滞的子弹波纹；大卫消散时露西悬在半空的嘴唇；丽贝卡消散前洒脱的笑容。
    互动细节：丽贝卡一把将大卫推向露西的动作；露西跨坐在大卫腿上挽留。

- 必然触发台词/梗：
    - 梦魇亚当重锤：“浪费时间的玩闹。你们是不可能击败我的。”
    - 丽贝卡：“大卫就交给你了。跑起来，露西！”
    - 大卫：“我只是你的梦。而现在……梦该醒了。”
    - 露西：“我还有很多的话想和你说。” / 大卫：“我也是。不过，我相信现在的你已不需要那些安慰的话了。”
    - 丽贝卡：“别露出那种表情啦……我最受不了婆婆妈妈的告别了。拜拜笨蛋，照顾好你自己。”

:: 主线事件6：星炬的余晖与梦的尽头 ::

> 事件概况
* 梦境危机解除后，星炬学院恢复了日常。在离开前的这几天里，露西留下来协助莫宁处理善后工作。
* 露西来到了之前那面出现数据化裂缝的墙壁前。裂缝已经愈合，但琳奈、西格莉卡和丽贝卡共同完成的涂鸦依然色彩鲜艳。
* 离别的时刻终于到来。白色守门人提醒露西，穿过最深处的门扉回到原世界后，在索拉里斯的这段经历会被当作梦境逐渐遗忘。它还暗示心之集域其实有让她永远留在这里的方法。
* 露西微笑着婉拒了这份提议，表示虽然有些可惜，但回到现实是她与同伴们共同的愿望。
* 在月球心之集域门扉前，露西与漂泊者独处。漂泊者询问她是否好好道了别，露西坦言自己原本只打算安静离开，但多亏了那只猫多管闲事的列车才让她没有留下遗憾，她其实很感激这段旅程。
* 露西将洛瑟菈校长赠送的纪念品，交托给漂泊者保管，因为她无法将实体物品带回自己的世界。
* 门扉的光芒开始闪烁。露西转过身，用极其温柔的语气向漂泊者道出最后的告别。在漂泊者的注视下，她留下一句意味深长的“让我们在梦中相见”后，踏入了光芒之中，彻底离开了索拉里斯。
* 漂泊者拿起纪念品，发现那是洛瑟菈通过读取露西记忆留下的照片——大卫、露西、丽贝卡、漂泊者、阿布，所有人都在月球上洋溢着笑容。

> 区块目标
* 目标1：详细交代露西这几天在学院的行动，展现她从抑郁中走出来后愿意帮助他人、感受日常美好的积极转变。
* 目标2：通过涂鸦与学生训练的细节，呼应前文情节，完成星炬学院支线剧情的圆满收束。
* 目标3：聚焦露西与漂泊者的告别，通过交托礼物和面对遗忘的坦然，将故事的情感余韵推向最高潮。

> 执行引导
* 描写重点：
    视觉镜头：墙面上色彩斑斓的涂鸦与阳光明媚的校园；露西将礼物递给漂泊者时依依不舍却又坚定的眼神；门扉开启时吞没露西背影的柔和白光。
    互动细节：露西手指轻轻拂过墙上八哥犬涂鸦的动作；漂泊者郑重接过礼物并点头承诺的默契；露西跨入门扉前回头留下的那个毫无阴霾的微笑。

* 必然触发台词/梗：
    * 露西：“只靠我一个人是无法做到的。你的验算提供了很多帮助……也算是个有趣的研究素材吧？”
    * 白色守门人：“在心之集域的某处，其实存在着让你留下来的方法……”
    * 露西：“谢谢。不过，我还是想要回去。这是他们的愿望，也是我的……愿望。”
    * 露西：“洛瑟菈送的礼物。我没办法带走它，还是交给你吧。”
    * 露西：“我喜欢这个世界，让我们……在梦中相见吧。”
