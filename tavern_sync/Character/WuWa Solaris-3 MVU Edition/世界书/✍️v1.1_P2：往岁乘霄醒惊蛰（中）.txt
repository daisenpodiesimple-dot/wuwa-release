:: 主线事件1：密室辛夷与心核之谜 ::

> 事件概况
- 追踪痕迹至虹镇瀑布后的密室，见到虹镇负责人辛夷（女性长者，今汐的老师）与避难的乡民。
- 辛夷告知今汐已前往“叩天关”（稷廷遗址）寻找开启眠龙庭的钥匙“心核”。
- 辛夷转述今汐的决意：岁主预言必有一战，她打算在解救岁主后直面这个预言。
- 辛夷解释“眠龙庭”是岁主的休眠舱，也揭示了虹镇人留守此地守护岁主的宿命。
- 长离决定前往叩天关与今汐汇合，并以此为饵诱捕残星会主力。
- 辛夷补充道，今汐在离开前特意安排了密室的防守，即使身处险境，她依然将民众的安危放在首位。
- 辛夷感叹今汐与乘霄山的缘分早已注定，仿佛这一切都是命运的轮回，而今汐正是那个破局之人。

> 区块目标
- 目标1：通过辛夷之口补全背景设定（虹镇、眠龙庭）。
- 目标2：侧面描写今汐即便虚弱却依然顾全大局的领袖气质。

> 执行引导
- 描写重点：
  密室中乡民们不安却信任的眼神；辛夷谈起今汐时既敬佩又担忧的语气。
- 必然触发台词/梗：
  - 辛夷：“‘叩天关’是虹镇人的叫法，也许，你们更熟悉‘稷廷遗址’这个名字。”
  - 辛夷：“待岁主安然无恙地脱困后，我必须认真地与祂聊一聊这所谓的预言。”
  - 辛夷：“自乘霄山而来，而今重归乘霄山，令尹与此地的机缘，早已注定……”

:: 主线事件2：叩天遗址与拟像守卫 ::

> 事件概况
- 击碎崖壁机关开启叩天关大门，进入遗址。
- 遭遇由能量构成的“拟像”怪物，长离识破这是稷廷的防卫机制。
- 在遗迹中层发现“曜纹”机关门，需要寻找三处“曜岩”对应开启。
- 在石门前发现今汐留下的血迹与脱落的龙鳞，长离判断今汐已接近“超频”界限，情况危急。
- 漂泊者与长离在遗迹中配合默契，利用溯流仪和高能黑石解开了一系列复杂的稷廷机关，展现了漂泊者对古代科技的某种本能熟悉。
- 长离看着地上的血迹，眼神中流露出痛惜，她明白今汐正在透支生命来争取时间。

> 区块目标
- 目标1：展示叩天关遗址独特的稷廷科技风格（曜纹、拟像）。
- 目标2：通过血迹与龙鳞，将紧张感推向高潮。

> 执行引导
- 描写重点：
  拟像怪物半透明的虚幻质感；地面上那抹触目惊心的殷红血迹与微光黯淡的龙鳞。
- 必然触发台词/梗：
  - 长离：“这血迹和鳞片……是今汐的……更有可能是，她已经接近**超频**的界限。”
  - 长离：“石门上有星辰流光跃动，是「曜纹」。”
  - 长离：“留给我们的时间不多了。”

:: 主线事件3：稷廷卷宗与二次共鸣 ::

> 事件概况
- 在探索中发现稷廷卷宗，揭露岁主降临之初便身负重伤，失去了复原时流的能力。
- 卷宗记载了疯狂的“二次共鸣”实验：通过共鸣力碰撞，试图人为制造时序之能。
- 只有一次实验成功，虽然「溯流仪」最终崩毁，但验证了“超频”是二次共鸣的前提。
- 长离注意到卷宗提到岁主曾听从某位“奇人”的指令传输能量，暗示漂泊者就是那个关键人物。
- 卷宗中还记录了稷廷学者对“二次共鸣”理论的痴迷，他们认为这是凡人触碰神明权柄的唯一途径，哪怕代价是毁灭。
- 长离推测今汐正是看到了这份卷宗，才决定效仿实验中的方法，通过与岁主的对抗来达成二次共鸣，以此修复岁主的时序之能。

> 区块目标
- 目标1：为后续“命定之战”的机制（碰撞、超频）做理论铺垫。
- 目标2：进一步确认漂泊者身为“御者/奇人”的特殊身份。

> 执行引导
- 描写重点：
  长离翻阅卷宗时指尖在关键文字上的停顿；漂泊者听到“奇人”描述时的恍然大悟。
- 必然触发台词/梗：
  - 长离：“频率的碰撞或可引发二次共鸣……即是彼此共鸣的两方力量的对抗与冲撞。”
  - 长离：“那个实验人员认为，超频，或许是二次共鸣的前提条件。”
  - 长离：“在那次例外中……那位岁主身侧的‘奇人’似对祂下达了一条指令。”

:: 主线事件4：师徒重逢与生死辩论 ::

> 事件概况
- 解开机关后，在遗迹深处找到虚弱的今汐。
- 今汐坦白打算利用二次共鸣，通过与岁主交锋来回正时流，哪怕代价是牺牲自己。
- 长离温柔地劝诫今汐，指出这是在自寻死路，并不能长久地守护今州。
- 今汐反驳这是为了效仿老师（长离）不惜命的精神。长离软化态度，表示活着才能做更多。
- 长离将漂泊者推到台前，指出漂泊者是除了牺牲之外的唯一变数与希望。
- 今汐看着长离，眼中含泪，她虽惜命，但为了守护今州，她愿意付出一切，这种决绝让长离既心疼又无奈。
- 漂泊者坚定地站在今汐身边，承诺会与她共同面对，这让今汐在绝境中感受到了温暖和依靠。

> 区块目标
- 目标1：爆发全篇最强烈的情感冲突，展现师徒情深与理念分歧。
- 目标2：确立漂泊者在两人关系中的调停者与破局者地位。

> 执行引导
- 描写重点：
  今汐苍白却倔强的脸庞；长离从严厉转为无奈再到温情的眼神变化。
- 必然触发台词/梗：
  - 今汐：“若我一战，不仅能保下岁主，还能保下今州。代价，只需我一人。”
  - 长离：“生死虽有常，可活着，能做更多。”
  - 长离：“漂泊者本就与此地有极深的渊源，这一程若想破局，你们二人，缺一不可。”
  - 今汐：“能被你这样信任着，真好。”

:: 主线事件5：烈焰断后与前路独行 ::

> 事件概况
- 三人合力开启中枢，取得散发幽蓝光芒的“心核”。
- 残星会追兵（造匠）逼近，长离主动转身面向敌人，背对今汐与漂泊者。
- 长离表示她的引路任务已完成，接下来是漂泊者与今汐的路，她将作为棋子留在最合适的位置。
- 长离拔出佩剑，赤红色的流火在剑身燃起，独自一人挡住千军万马。
- 漂泊者拉着不舍的今汐，带着心核向另一侧出口撤离，身后是冲天的火光。
- 长离在火光中回头一笑，那笑容既是告别也是鼓励，她相信漂泊者和今汐一定能完成使命。
- 残星会造匠们试图围攻长离，但被长离精准而凌厉的剑术瞬间击退。
- 长离独自断后击退残星会追兵，漂泊者与今汐带着心核撤离叩天关。

> 区块目标
- 目标1：塑造长离高光时刻，展现其强大的实力与从容的气度。
- 目标2：完成队伍交接，剧情焦点转回今汐与漂泊者。

> 执行引导
- 描写重点：
  长离背影的孤傲与决绝；赤色火焰在幽暗遗迹中炸裂的视觉冲击；今汐频频回头的泪光。
- 必然触发台词/梗：
  - 长离：“别忘了，我是你的‘引路人’……自此地往后，便是你与今汐一起，去往「角」的身前。”
  - 长离：“啊呀，人这就来了……不如由我来教教你们瑝珑的礼数。”
  - 长离：“促成这一局，即是我能做的所有。”
