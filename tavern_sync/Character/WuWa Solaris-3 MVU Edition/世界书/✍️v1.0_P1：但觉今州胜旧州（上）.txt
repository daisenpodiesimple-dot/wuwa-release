:: 主线事件1：星海坠落与云陵谷苏醒 ::

> 事件概况
- 意识悬浮于浩瀚星海之中，一位身姿朦胧的神秘存在（女神）注视着漂泊者。她将凝聚的力量注入青年胸口，力量经握手完成交接，声波状刻印浮现于青年右手手背。
- 女神随即将青年推入未知浪潮。青年苏醒后拼命游动冲出海面，却发现海面倒悬于空中，这便是天空海。随后失去意识。
- 黑衣青年在云陵谷醒来，头枕在秧秧腿上。模糊且温柔的女声与元气的女声在旁交谈，讨论他口鼻无异物却有呛水反应的症状。秧秧发现青年醒了。
- 炽霞与秧秧表明身份，此处是云陵谷，瑝珑属下今州城的边地野外。青年记忆全无，装束不像本地人。白芷此前已对昏迷的漂泊者做过简单诊疗。
- 秧秧以风感知到天空海聚拢、无音区即将生成，葫芦受影响失去信号，需尽快与白芷会合撤离。秧秧以"漂泊者"暂称失忆的青年，邀其同行出谷。
- 离谷途中炽霞据手背声痕与葫芦判定漂泊者是共鸣者。经过龙形雕像时，漂泊者恍惚间一只真正的龙形生物浮现眼前，与漂泊者对面而立。秧秧解释那是今州岁主「角」的雕像，今州岁主的共鸣者正是今州令尹。

> 区块目标
- 目标1：以女神推落与天空海坠落建立身世悬念，埋下"被选中、被赋予力量"的伏笔。
- 目标2：以岁主雕像共感引出与「角」的潜在共鸣，塑造秧秧、炽霞初始形象。

> 执行引导
- 描写重点：开场星海中女神的身姿朦胧与注视；漂泊者冲出倒悬海面时的失重与错愕；漂泊者枕在秧秧腿上醒来，秧秧低头看他时的温柔。
- 必然触发台词/梗：
  - 秧秧：「这里是云陵谷，瑝珑属下今州城的边地野外。」
  - 秧秧：「漂泊者……在记起原本的名字之前，可以这么称呼你吗？」
  - 秧秧：「今州的岁主名为『角』，龙形，就是这座雕像所呈现的模样。」
  - 炽霞：「下次我必定一马当先！不然我就不叫马小芳……不对不对，不叫炽霞！」

:: 主线事件2：无音区遭遇战与肉身吸收 ::

> 事件概况
- 出谷后众人发现悬崖下无音区突然生成，白芷在中央探查。秧秧解释刚形成的无音区有"蛰伏期"，短期内相对安全，众人展开终端存储的伞翼降落进入。
- 变故突生，一道屏障将炽霞、白芷隔在外。漂泊者与秧秧遭遇无冠者，一头具象化了战争行为本身的强大残象。
- 漂泊者与秧秧合力击溃无冠者，残象原地留下金色虚影（残响）。炽霞让漂泊者用终端吸收，却发现终端无法回收。
- 一阵突兀的「饿……好饿……」从漂泊者右手手背的声痕内传来，残响的频率随即被直接吸入声痕之内。炽霞震惊，白芷指出肉身吸收声骸并非完全没有先例。
- 据瑝珑典籍，瑝珑创立前曾有"天人"吸收最强残象、重塑世间，此为瑝珑历史开端。漂泊者的能力与传说如出一辙。

> 区块目标
- 目标1：以战斗揭示漂泊者肉身吸收声骸的核心异常能力。
- 目标2：借"天人"典籍把身份悬念上溯至瑝珑起源。

> 执行引导
- 描写重点：屏障隔开众人时的孤立感；漂泊者赤手吸收残响频率时炽霞瞠目的反应；金色虚影悬在原地、终端无法回收的落差。
- 必然触发台词/梗：
  - 秧秧：「我从没见过这么厉害的残象，但漂泊者却能将他完全击溃。」
  - 声痕内的声音：「饿……好饿……」
  - 炽霞：「……斯天人降世，乾坤指掌，纳骸于身；一气凝化，盘古生焉……」

:: 主线事件3：入城与边庭三日之约 ::

> 事件概况
- 信号恢复后，一则面向今州全域的影像留言传来。今州令尹今汐称正等候一位重要的客人，请对方于边庭一见。众人隐约感觉今汐所等之人就是漂泊者。
- 进城前漂泊者有种被注视的感觉。一名神秘少女（椿，黑海岸执花，漂泊者不知其身份）在隐秘处默默关注，留下"……命·定·之·种"的低语。
- 炽霞领漂泊者到中枢信标登记终端。边庭前聚集大量求见者，均被令尹近卫散华一眼劝返。
- 散华一眼认出漂泊者正是今汐所等之人。今汐不在边庭，留下四件信物（糖丸、日晷、树叶、莽吉柿），以三日为期。
- 散华说自己能看见生物本身的频率，且只能看到"频率"。漂泊者与今汐的频率是一样的：没有被扭曲的、真实的、原本的姿态。散华授予漂泊者通行权限。出边庭后秧秧感应到风中动荡的气息。

> 区块目标
- 目标1：以今汐广播正式引入令尹线，以"频率相同"将漂泊者与今汐命运勾连。
- 目标2：以三日之约和四件信物建立解谜驱动；以椿的注视埋下暗线悬念。

> 执行引导
- 描写重点：今汐影像留言中少女令尹的沉稳与恳切；散华注视漂泊者时眼神的精准，她的视线落在频率上；四件信物摆在桌面上的寻常与隐秘。
- 必然触发台词/梗：
  - 今汐：「在到来的客人中，有一位对我、对今州、乃至瑝珑都很重要的客人。我们已等候许久。」
  - 散华：「您与……令尹大人的频率是一样的。」
  - 散华：「以三日为期，令尹定当与您正式会面。」

:: 主线事件4：信物调查与糖丸线索 ::

> 事件概况
- 漂泊者向秧秧、炽霞展示信物。炽霞认出糖丸是小时候吃过如今已不见的款式，秧秧回忆幼时打针后护士奖励过同样的糖。日晷晷面刻度特别，树叶有残象频率波动，莽吉柿非今州本土、靠水陆货运。
- 秧秧推演日晷：外圈十二时辰、中央四角四神兽方位，但缺太阳光下日晷无用，暗示指向的并非计时本身。
- 莫特斐解析日晷是空心榫卯机关盒，内有夹层存纸质卷轴，缺关键部件，组装配件在相里要工作室。白芷为漂泊者做体检：生理健康、共鸣频率无异常，波形稳定性比她见过的共鸣者都高，体内似存在另一个空间能汲取频率能量。
- 模拟领域检测中，漂泊者描述出"手持镰刀的巨大残象"与"虚实变幻的巨大月亮"。白芷判断可能是潜意识与模拟领域共感后的折射。
- 白芷检测出糖丸并非糖果，而是一颗过期二十年的口服疫苗制剂，指向二十多年前的一场今州儿童疫病。二十年前那一年正是干支纪法的未年，与日晷上的时间密码「未」对应。
- 众人据此前往公共档案室，以「未」字列检索到两张照片：一张是研究员合影（领头是一位三十岁左右的女性），一张是一只手握着汤勺喂糖丸给襁褓中婴儿的画面。报道记载首席在糖丸研发完成三年后病逝。秧秧注意到婴儿与漂泊者年龄相仿但并非金瞳，推测糖丸只是线索一环。
- 漂泊者从资料中了解今州建州史：今州建州于未年，云陵谷一役是建州契机，无冠者的出现改变了残象潮原本路径，与鸣式有联系。

> 区块目标
- 目标1：四件信物各自建立线索指向，以白芷体检佐证漂泊者身体异常。
- 目标2：借糖丸线索（疫苗→未年→档案室）完成今州建州史与无冠者/鸣式关系的背景补全。

> 执行引导
- 描写重点：四件信物摊开时的日常感与隐秘感；白芷检测时异常稳定却无法解释的波形图；漂泊者推演信物时秧秧追问补全的节奏。
- 必然触发台词/梗：
  - 莫特斐：「我直接说结论吧，它是空心的。你听，轻轻叩击外壳还可以听见回响。」
  - 白芷：「你的波形检测图显示稳定性很高，比我见过的共鸣者都高，你的超频风险极低。」
  - 秧秧：「无冠者是具象化了战争行为本身的残象——战死者的怨恨，偷生者的恐惧，构成了无冠者的全部。」

:: 主线事件5：出城遇鉴心与伏波营见闻 ::

> 事件概况
- 翌日在攀花饭馆汇合。秧秧从树叶上感知到残象频率波动，漂泊者描述出梦中星海、黑发少女、倒悬天空海的景象，线索指向城外。
- 城门戒备增强，小道士鉴心因无权限被拦在城门口，这是她首次出场。她受退役军士宗业之托出城寻其孙儿致远，漂泊者以权限带她同行出城。
- 途中遇伪装成平民的流放者破坏侦察塔零件。鉴心给食物和钱，对方却恩将仇报想抢劫，被制服后自述是粮钱不够才偷零件。鉴心劝他们回城找活，留下"六羡茶馆寻鉴心"的话，随后修好两座失联侦察塔。
- 抵达伏波营，军需官休兰正忙于物资调度，前线伤员陆续运下。逞强的穗禾能自愈却不愿占用医疗资源，众人协助医护人员完成治疗。医护提及忌炎将军改制成立夜归军后伤员状况已好转。军需官错把莽吉柿看成榴弹，由此讲起黑石武器演进与补给的意义。
- 鉴心找到致远，告知其爷爷宗业已到今州寻他。鉴心决定留在伏波营，引师父之言"上士得道于三军"。致远受启发决定回城为战事出力。众人在伏波营外告别，鉴心自此离队，漂泊者与秧秧继续追查树叶指向。

> 区块目标
- 目标1：以伏波营见闻把"莽吉柿→战争"的信物线索落地，让漂泊者具象感知前线战事。
- 目标2：以鉴心"三军得道"的转变完成角色成长起点，以致远回城出力呼应"各尽其能"。

> 执行引导
- 描写重点：鉴心给流放者食物和钱时毫无犹豫的坦荡；伏波营里伤员与物资的紧迫感；穗禾嚷着要回前线的倔强。
- 必然触发台词/梗：
  - 鉴心：「师父说过，常善救人，故无弃人。所以，不必客气！」
  - 鉴心：「上士得道于三军，中士得道于都市，下士得道于山林。」
  - 致远：「我和共鸣者比不了，继续待下去可能确实是个拖油瓶，但是回城之后我能做的事就多了！」
  - 鉴心：「一路顺风，有缘再见。」
